---
title: 'Actividad'
openapi: '/es/api-reference/v2-openapi.json GET /team/activity'
---

Muestra tu actividad reciente de la API en las últimas 24 horas. Úsalo para identificar IDs de trabajo y luego recuperar los resultados con el endpoint GET correspondiente.

| Endpoint       | Endpoint de recuperación    |
| -------------- | --------------------------- |
| `scrape`       | `GET /v2/scrape/{id}`       |
| `crawl`        | `GET /v2/crawl/{id}`        |
| `batch_scrape` | `GET /v2/batch/scrape/{id}` |
| `agent`        | `GET /v2/extract/{id}`      |
