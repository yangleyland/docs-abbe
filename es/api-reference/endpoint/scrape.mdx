<div id="interactions">
  ## Interacciones
</div>

Para las interacciones del navegador (hacer clic, escribir, navegar, extraer contenido dinámico), usa el [endpoint de Interact](/es/features/interact). Primero haz scraping de una página y luego llama a `POST /v2/scrape/{scrapeId}/interact` con un prompt en lenguaje natural o código de Playwright para realizar acciones en la página.

Consulta la [documentación de Interact](/es/features/interact) para ver todos los detalles y ejemplos.

Opcionalmente, también puedes usar el parámetro `actions`, aunque no se recomienda usarlo para interacciones complejas.
