Firecrawl es de código abierto y está disponible bajo la [licencia AGPL-3.0](https://github.com/firecrawl/firecrawl/blob/main/LICENSE). Para ofrecer el mejor producto posible, también ofrecemos una versión en la nube alojada en [firecrawl.dev](https://firecrawl.dev) con funciones adicionales e infraestructura gestionada.

<div id="feature-comparison">
  ## Comparación de funcionalidades
</div>

La versión de código abierto incluye el motor principal de scraping y crawling. Firecrawl Cloud añade capacidades a nivel de infraestructura, un panel de gestión y funciones empresariales, además de todo lo incluido en la edición de código abierto.

![Firecrawl Cloud vs Open Source](/images/open-source-cloud.png)

| Función | Open Source | Cloud |
|---------|:----------:|:-----:|
| Scrape | ✔ | ✔ |
| Crawl | ✔ | ✔ |
| Map | ✔ | ✔ |
| Search | ✔ | ✔ |
| Scrape por lotes | ✔ | ✔ |
| Extract | ✔ | ✔ |
| Modo JSON | ✔ | ✔ |
| Formatos listos para LLM | ✔ | ✔ |
| Control de cambios | ✔ | ✔ |
| SDKs | ✔ | ✔ |
| Agent | ✘ | ✔ |
| Sandbox de navegador | ✘ | ✔ |
| Acciones | ✘ | ✔ |
| Proxies mejorados | ✘ | ✔ |
| Rotación de proxies | ✘ | ✔ |
| Panel | ✘ | ✔ |
| Funciones empresariales | ✘ | ✔ |

<div id="why-both">
  ## ¿Por qué ambos?
</div>

La solución en la nube nos permite innovar de forma continua y mantener un servicio de alta calidad y sostenible para todos los usuarios. Los ingresos de Firecrawl Cloud financian el desarrollo continuo del proyecto de código abierto.

* **Código abierto** — Ejecuta Firecrawl en tu propia infraestructura con control total sobre el motor de scraping principal. Consulta [Ejecución local](/es/contributing/guide) y [Autoalojamiento](/es/contributing/self-host).
* **Nube** — Empieza de inmediato con un servicio gestionado que incluye proxies mejorados, un panel de uso y funciones empresariales. Regístrate en [firecrawl.dev](https://firecrawl.dev).
