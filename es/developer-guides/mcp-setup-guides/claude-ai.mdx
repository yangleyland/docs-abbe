Añade capacidades de scraping web y búsqueda a Claude.ai con Firecrawl MCP mediante conectores personalizados.

<Note>
  ¿Buscas la configuración de Claude Code? Consulta la [guía de Claude Code
  ](/es/quickstarts/claude-code) en su lugar.
</Note>

<div id="quick-setup">
  ## Configuración rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtén tu clave de API
</div>

Regístrate en [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys) y copia tu clave de API.

<div id="2-add-custom-connector">
  ### 2. Agregar conector personalizado
</div>

Ve a [Configuración &gt; Conectores](https://claude.ai/settings/connectors) en Claude.ai y haz clic en **Agregar conector personalizado**.

Completa los datos del conector:

* **URL:** `https://mcp.firecrawl.dev/YOUR_API_KEY/v2/mcp`
* **OAuth Client ID:** Déjalo en blanco
* **OAuth Client Secret:** Déjalo en blanco

Reemplaza `YOUR_API_KEY` en la URL por tu [clave de API de Firecrawl](https://www.firecrawl.dev/app/api-keys). Tu clave de API se incluye directamente en la URL, por lo que no necesitas campos de autenticación adicionales.

Haz clic en **Agregar** para guardar el conector.

<Tip>
  **¿Prefieres no incluir tu clave de API en la URL?** Usa el endpoint sin clave `https://mcp.firecrawl.dev/v2/mcp` en su lugar (dejando igualmente los campos de OAuth en blanco). Claude.ai abrirá una ventana del navegador para que inicies sesión en Firecrawl y autorices el conector. Consulta [Connect MCP Clients with OAuth](/es/developer-guides/mcp-setup-guides/oauth) para más detalles.
</Tip>

<div id="3-enable-in-conversation">
  ### 3. Actívalo en la conversación
</div>

En cualquier conversación de Claude.ai, haz clic en el botón **+** de la esquina inferior izquierda, ve a **Conectores** y activa el conector de Firecrawl.

<div id="quick-demo">
  ## Demostración rápida
</div>

Con el conector de Firecrawl habilitado, prueba con estos prompts:

**Buscar en la web:**

```
Search for the latest Next.js 15 features
```

**Hacer scraping de una página:**

```
Scrape firecrawl.dev and tell me what it does
```

**Consulta la documentación:**

```
Find and scrape the Stripe API docs for payment intents
```

Claude usará automáticamente las herramientas de búsqueda y scraping de Firecrawl para obtener la información.
