Añade las herramientas de búsqueda, scraping, rastreo y navegador de Firecrawl a [Sourcegraph Amp](https://ampcode.com) mediante MCP.

<div id="quick-setup">
  ## Configuración rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtén tu clave de API
</div>

Regístrate en [firecrawl.dev/app](https://www.firecrawl.dev/app) y copia tu clave de API.

<div id="2-add-firecrawl-to-amp">
  ### 2. Añade Firecrawl a Amp
</div>

Abre la configuración de Amp y añade un MCP Server. Amp admite la configuración estándar de MCP:

```json
{
  "amp.mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Sustituye `fc-YOUR-API-KEY` por tu clave de API de Firecrawl.

<div id="3-reload-amp">
  ### 3. Vuelve a cargar Amp
</div>

Vuelve a cargar la ventana de Amp. Las herramientas de Firecrawl ya están disponibles para el agente.

<div id="remote-hosted-url-no-nodejs-required">
  ## URL alojada remotamente (sin necesidad de Node.js)
</div>

```json
{
  "amp.mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Solución de problemas
</div>

* **El servidor no se inicia** — revisa la vista de registros de MCP de Amp para ver la salida de stderr.
* **Falta `npx`** — instala Node.js 18+ o usa la URL alojada remotamente indicada arriba.
