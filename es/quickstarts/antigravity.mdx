Añade las herramientas de búsqueda, scraping, rastreo y navegador de Firecrawl a [Google Antigravity](https://antigravity.google/) mediante MCP.

<div id="quick-setup">
  ## Configuración rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtén tu clave de API
</div>

Regístrate en [firecrawl.dev/app](https://www.firecrawl.dev/app) y copia tu clave de API.

<div id="2-add-firecrawl-to-antigravity">
  ### 2. Añade Firecrawl a Antigravity
</div>

Abre la configuración de Antigravity (`Cmd/Ctrl + ,`), busca **MCP Servers** y añade un servidor nuevo con:

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Reemplaza `fc-YOUR-API-KEY` por tu clave de API de Firecrawl.

<div id="3-reload-antigravity">
  ### 3. Vuelve a cargar Antigravity
</div>

Vuelve a cargar la ventana (`Cmd/Ctrl + Shift + P` → `Reload Window`). El agente ahora tiene disponibles las herramientas web de Firecrawl.

<div id="quick-demo">
  ## Demostración rápida
</div>

En un chat con el agente de Antigravity:

```
Search the web for "Vercel AI SDK v5 release notes" and summarize.
```

```
Scrape https://docs.firecrawl.dev/ai-onboarding and list every linked guide.
```

```
Crawl https://example.com and extract every page title.
```

Antigravity dirige automáticamente esas llamadas a herramientas a través de Firecrawl MCP.

<div id="remote-hosted-url-no-nodejs-required">
  ## URL alojada remotamente (sin necesidad de Node.js)
</div>

```json
{
  "mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Solución de problemas
</div>

* **El servidor aparece como &quot;failed&quot;** — revisa el panel de salida de MCP para ver `stderr`. La mayoría de los errores se deben a que falta una clave de API o a que `npx` no está en el `PATH`.
* **Las herramientas no se ejecutan** — menciona explícitamente Firecrawl (por ejemplo, &quot;Usa Firecrawl para hacer scraping…&quot;) en tu primer prompt para que el agente seleccione la herramienta correcta.
