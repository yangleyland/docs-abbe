<div id="prerequisites">
  ## Prerrequisitos
</div>

* Bun 1.0+
* Una clave de la API de Firecrawl — [obtén una gratis](https://www.firecrawl.dev/app/api-keys)

<div id="install-the-sdk">
  ## Instala el SDK
</div>

```bash
bun add firecrawl
```

Añade tu clave de API a `.env`:

```bash
FIRECRAWL_API_KEY=fc-YOUR-API-KEY
```

<div id="search-the-web">
  ## Buscar en la web
</div>

Bun tiene un servidor HTTP integrado. Crea `index.ts`:

```typescript
import { Firecrawl } from "firecrawl";

const firecrawl = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

Bun.serve({
  port: 3000,
  async fetch(req) {
    const url = new URL(req.url);

    if (req.method === "POST" && url.pathname === "/search") {
      const { query } = await req.json();
      const results = await firecrawl.search(query, { limit: 5 });
      return Response.json(results);
    }

    return new Response("Not found", { status: 404 });
  },
});

console.log("Server running on port 3000");
```

Ejecútalo:

```bash
bun run index.ts
```

<div id="scrape-a-page">
  ## Haz scraping de una página
</div>

Agrega una ruta `/scrape` al mismo servidor:

```typescript
if (req.method === "POST" && url.pathname === "/scrape") {
  const { url: targetUrl } = await req.json();
  const result = await firecrawl.scrape(targetUrl);
  return Response.json(result);
}
```

<div id="interact-with-a-page">
  ## Interactúa con una página
</div>

Usa interact para controlar una sesión activa del navegador: haz clic en botones, completa formularios y extrae contenido dinámico.

```typescript
if (req.method === "POST" && url.pathname === "/interact") {
  const { url: targetUrl } = await req.json();

  const result = await firecrawl.scrape(targetUrl, { formats: ['markdown'] });
  const scrapeId = result.metadata?.scrapeId;

  await firecrawl.interact(scrapeId, { prompt: 'Search for iPhone 16 Pro Max' });
  const response = await firecrawl.interact(scrapeId, { prompt: 'Click on the first result and tell me the price' });

  await firecrawl.stopInteraction(scrapeId);

  return Response.json({ output: response.output });
}
```

<div id="script-usage">
  ## Uso del script
</div>

Usa Firecrawl en un script de Bun independiente:

```typescript
import { Firecrawl } from "firecrawl";

const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });
const results = await app.search("firecrawl web scraping", { limit: 5 });
console.log(results);
```

```bash
bun run search.ts
```
