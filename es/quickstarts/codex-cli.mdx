Añade a [OpenAI Codex CLI](https://github.com/openai/codex) las herramientas de búsqueda, scraping, rastreo y navegador de Firecrawl mediante MCP.

<div id="quick-setup">
  ## Configuración rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtén tu clave de API
</div>

Regístrate en [firecrawl.dev/app](https://www.firecrawl.dev/app) y copia tu clave de API.

<div id="2-add-firecrawl-to-codex">
  ### 2. Añade Firecrawl a Codex
</div>

Codex lee la configuración del MCP Server desde `~/.codex/config.toml`. Añade el servidor de Firecrawl:

```toml
[mcp_servers.firecrawl]
command = "npx"
args = ["-y", "firecrawl-mcp"]

[mcp_servers.firecrawl.env]
FIRECRAWL_API_KEY = "fc-YOUR-API-KEY"
```

Reemplaza `fc-YOUR-API-KEY` por tu clave de API de Firecrawl.

<div id="3-start-codex">
  ### 3. Inicia Codex
</div>

```bash
codex
```

Codex detecta las herramientas de Firecrawl al iniciarse. Comprueba que estén cargadas:

```bash
/mcp
```

Deberías ver `firecrawl` en la lista, con herramientas como `firecrawl_search`, `firecrawl_scrape`, `firecrawl_crawl` y `firecrawl_extract`.

<div id="quick-demo">
  ## Demostración rápida
</div>

Prueba estos prompts:

```
Busca en la web las notas de la versión más reciente de Next.js App Router y resúmelas.
```

```
scraping https://docs.firecrawl.dev and list the top-level sections.
```

```
rastreo https://example.com and save the markdown for every page under /blog.
```

<div id="remote-hosted-url-no-nodejs-required">
  ## URL alojada remotamente (sin necesidad de Node.js)
</div>

Si prefieres no ejecutar `npx` localmente:

```toml
[mcp_servers.firecrawl]
url = "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
```

<div id="troubleshooting">
  ## Solución de problemas
</div>

* **Codex no detecta las herramientas** — ejecuta `codex --version` para confirmar que estás usando una versión compatible con MCP y, después de editar `config.toml`, reinicia la CLI.
* **`spawn npx ENOENT`** — instala Node.js 18+ y asegúrate de que `npx` esté en tu `PATH`, o cambia a la URL alojada remotamente de arriba.
* **401 / clave no válida** — vuelve a generar una clave de API en [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys).
