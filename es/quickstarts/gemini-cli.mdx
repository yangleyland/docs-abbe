Añade las herramientas de búsqueda, scraping, rastreo y navegador de Firecrawl a [Google Gemini CLI](https://github.com/google-gemini/gemini-cli) mediante MCP.

<div id="quick-setup">
  ## Configuración rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtén tu clave de la API
</div>

Regístrate en [firecrawl.dev/app](https://www.firecrawl.dev/app) y copia tu clave de la API.

<div id="2-add-firecrawl-to-gemini-cli">
  ### 2. Añade Firecrawl a Gemini CLI
</div>

Gemini CLI lee la configuración de MCP desde `~/.gemini/settings.json` (global) o desde `.gemini/settings.json` en tu proyecto. Añade:

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Sustituye `fc-YOUR-API-KEY` por tu clave de la API de Firecrawl.

<div id="3-launch-gemini-cli">
  ### 3. Inicia la Gemini CLI
</div>

```bash
gemini
```

Confirma que el servidor se haya cargado:

```
/mcp list
```

Deberías ver `firecrawl` y sus herramientas.

<div id="remote-hosted-url-no-nodejs-required">
  ## URL alojada remotamente (sin necesidad de Node.js)
</div>

```json
{
  "mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Solución de problemas
</div>

* **Las herramientas no aparecen** — reinicia Gemini CLI después de editar `settings.json`; los servidores MCP se cargan al iniciarse.
* **`spawn npx ENOENT`** — instala Node.js 18+ o usa la URL alojada remotamente.
* **Límite de tasa excedido** — actualiza tu plan de Firecrawl en [firecrawl.dev/pricing](https://www.firecrawl.dev/pricing).
