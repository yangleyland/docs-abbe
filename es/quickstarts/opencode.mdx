Añade a [OpenCode](https://opencode.ai) las herramientas de búsqueda, scraping, rastreo y navegador de Firecrawl mediante MCP.

<div id="quick-setup">
  ## Configuración rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtén tu clave de la API
</div>

Regístrate en [firecrawl.dev/app](https://www.firecrawl.dev/app) y copia tu clave de la API.

<div id="2-add-firecrawl-to-opencode">
  ### 2. Añade Firecrawl a OpenCode
</div>

OpenCode lee la configuración desde `~/.config/opencode/config.json` (global) o desde `./opencode.json` en tu proyecto. Añade:

```json
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "firecrawl": {
      "type": "local",
      "command": ["npx", "-y", "firecrawl-mcp"],
      "environment": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Reemplaza `fc-YOUR-API-KEY` por tu clave de la API de Firecrawl.

<div id="3-start-opencode">
  ### 3. Inicia OpenCode
</div>

```bash
opencode
```

OpenCode carga los servidores MCP al iniciarse. Confirma que Firecrawl esté conectado:

```
/mcp
```

<div id="remote-hosted-url-no-nodejs-required">
  ## URL alojada remotamente (no requiere Node.js)
</div>

```json
{
  "mcp": {
    "firecrawl": {
      "type": "remote",
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Solución de problemas
</div>

* **Servidor no conectado** — ejecuta `opencode doctor` para inspeccionar los errores de carga de MCP.
* **Permiso denegado con `npx`** — instala Node.js 18+ y asegúrate de que tu shell detecte la instalación (`which npx`).
