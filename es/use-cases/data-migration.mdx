Extrae contenido, estructura y metadatos de cualquier sitio web y transfórmalos para importarlos en una nueva plataforma. Firecrawl se encarga del scraping para que puedas centrarte en el mapeo de datos para tu sistema de destino.

<div id="start-with-a-template">
  ## Empieza con una plantilla
</div>

<Card title="Firecrawl Migrator" icon="github" href="https://github.com/firecrawl/firecrawl-migrator">
  Migra datos entre plataformas y sistemas de forma eficiente
</Card>

<div id="how-it-works">
  ## Cómo funciona
</div>

Dirige Firecrawl a tu sitio web de origen para rastrear cada página y devolver datos limpios y estructurados. A partir de ahí, transforma la salida para que coincida con el esquema de tu plataforma de destino e impórtala mediante la API de esa plataforma o sus herramientas de importación masiva.

<div id="what-you-can-migrate">
  ## Qué puedes migrar
</div>

* **Contenido**: Páginas, entradas, artículos, archivos multimedia, metadatos
* **Estructura**: Jerarquías, categorías, etiquetas, taxonomías
* **Usuarios**: Perfiles y datos de usuario cuando sean de acceso público
* **Ajustes**: Configuraciones, campos personalizados, flujos de trabajo
* **Comercio electrónico**: Productos, catálogos, inventario, pedidos

<div id="common-migration-use-cases">
  ## Casos de uso comunes de migración
</div>

Los usuarios crean herramientas de migración con Firecrawl para extraer datos de diversas plataformas:

<div id="faqs">
  ## Preguntas frecuentes
</div>

<AccordionGroup>
  <Accordion title="¿Cómo gestionan las migraciones a gran escala?">
    Nuestra infraestructura se escala automáticamente para manejar migraciones a gran escala. Admitimos procesamiento incremental con lotes y extracción en paralelo, lo que te permite migrar millones de páginas dividiéndolas en partes manejables, con seguimiento del progreso.
  </Accordion>

  <Accordion title="¿Puedo preservar el valor SEO durante la migración?">
    ¡Sí! Extrae todos los metadatos de SEO, incluidas las URL, los títulos y las descripciones, y aplica redirecciones adecuadas. Te ayudamos a mantener tu posicionamiento en buscadores durante la migración.
  </Accordion>

  <Accordion title="¿Qué pasa con los archivos multimedia y los adjuntos?">
    Firecrawl puede extraer y catalogar todos los archivos multimedia. Puedes descargarlos para volver a subirlos a tu nueva plataforma o referenciarlos directamente si mantienes el mismo CDN.
  </Accordion>

  <Accordion title="¿Cómo valido la migración?">
    Proporcionamos informes de extracción detallados y compatibilidad con herramientas de comparación. Puedes verificar la integridad del contenido, comprobar enlaces rotos y validar la integridad de los datos.
  </Accordion>

  <Accordion title="¿Puedo migrar contenido generado por usuarios y comentarios?">
    Sí, puedes extraer contenido generado por usuarios visible públicamente, incluidos comentarios, reseñas y publicaciones en foros. Los datos privados de usuarios requieren la autenticación y los permisos adecuados.
  </Accordion>
</AccordionGroup>
