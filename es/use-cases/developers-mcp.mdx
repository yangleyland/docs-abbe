Dale a tu asistente de programación con IA la capacidad de hacer scraping, rastrear y buscar en la web en tiempo real. El servidor MCP de Firecrawl se conecta a Claude Desktop, Cursor y otros clientes de Model Context Protocol para que tu asistente pueda consultar documentación actualizada, descubrir la estructura de los sitios y extraer datos estructurados bajo demanda.

<div id="how-it-works">
  ## Cómo funciona
</div>

Integra Firecrawl directamente en tu flujo de trabajo de programación con IA mediante Model Context Protocol. Una vez configurado, tu asistente de IA obtiene acceso a un conjunto de herramientas de web scraping que puede usar en tu nombre:

| Herramienta              | Qué hace                                                                 |
| ------------------------ | ------------------------------------------------------------------------ |
| **Scrape**               | Extrae contenido o datos estructurados de una sola URL                   |
| **extracción por lotes** | Extrae contenido de varias URL conocidas en paralelo                     |
| **mapeo**                | Descubre todas las URL indexadas de un sitio web                         |
| **Crawl**                | Recorre una sección de un sitio y extrae contenido de cada página        |
| **Search**               | Busca en la web y, de forma opcional, extrae contenido de los resultados |

Tu asistente elige automáticamente la herramienta adecuada. Pídele que &quot;lea la documentación de Next.js&quot; y hará scraping. Pídele que &quot;encuentre todas las publicaciones del blog en example.com&quot; y primero hará un mapeo y luego una extracción por lotes.

<div id="faqs">
  ## Preguntas frecuentes
</div>

<AccordionGroup>
  <Accordion title="¿Qué asistentes de IA son compatibles con MCP?">
    Actualmente, Claude Desktop y Cursor tienen compatibilidad nativa con MCP. Cada vez más asistentes de IA están añadiendo compatibilidad. También puedes usar el SDK de MCP para crear integraciones personalizadas.
  </Accordion>

  <Accordion title="¿Puedo usar MCP en VS Code u otros IDE?">
    VS Code y otros IDE pueden usar MCP mediante extensiones de la comunidad o integraciones de terminal. La compatibilidad nativa varía según el IDE. Consulta nuestro [repositorio de GitHub](https://github.com/firecrawl/firecrawl-mcp-server) para guías de configuración específicas por IDE.
  </Accordion>

  <Accordion title="¿Cómo puedo almacenar en caché documentos de acceso frecuente?">
    El servidor MCP almacena automáticamente en caché las respuestas durante 15 minutos. Puedes configurar la duración de la caché en la configuración de tu servidor MCP o implementar una lógica de caché personalizada.
  </Accordion>

  <Accordion title="¿Existe un límite de tasa para las solicitudes de MCP?">
    Las solicitudes de MCP usan tus límites de tasa estándar de la API de Firecrawl. Recomendamos agrupar solicitudes relacionadas y usar caché para documentación de acceso frecuente.
  </Accordion>

  <Accordion title="¿Cómo configuro MCP con mi clave de API de Firecrawl?">
    Sigue nuestra [guía de configuración](https://github.com/firecrawl/firecrawl-mcp-server#installation) para configurar MCP. Deberás añadir tu clave de API de Firecrawl al archivo de configuración de MCP. El proceso toma solo unos minutos.
  </Accordion>
</AccordionGroup>
