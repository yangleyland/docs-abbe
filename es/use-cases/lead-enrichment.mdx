<div id="how-it-works">
  ## Cómo funciona
</div>

1. **Apunta Firecrawl a una fuente.** Pasa a la API la URL de un directorio, el sitio web de una empresa o una lista de dominios.
2. **Extrae datos estructurados.** Firecrawl scrapea cada página y devuelve campos como el nombre de la empresa, los datos de contacto, los miembros del equipo y las noticias recientes.
3. **Envía los datos a tu CRM.** Envía registros enriquecidos a Salesforce, HubSpot o cualquier otra herramienta a través de la API o de una integración con Zapier.

<div id="keep-crm-data-fresh-automatically">
  ### Mantén los datos del CRM actualizados automáticamente

Firecrawl obtiene información en tiempo real directamente de los sitios web de las empresas, en lugar de una base de datos estática. Tu equipo siempre cuenta con las noticias más recientes de la empresa, cambios en el equipo y señales de crecimiento.

<div id="lead-sources">
  ## Fuentes de leads
</div>

Extrae leads de dos tipos de fuentes, según con qué datos ya cuentes.

<div id="business-directories">
  ### Directorios empresariales
</div>

Usa una URL de directorio cuando necesites una lista de empresas de un mercado o sector.

* Directorios específicos del sector
* Listados de cámaras de comercio
* Miembros de asociaciones sectoriales
* Listas de asistentes a conferencias

<div id="company-websites">
  ### Sitios web de empresas
</div>

Usa la URL de una empresa cuando ya conozcas la cuenta y quieras más contexto para la prospección.

* Páginas &quot;Acerca de&quot; y secciones del equipo
* Comunicados de prensa y noticias
* Ofertas de empleo como señales de crecimiento
* Casos de éxito de clientes

<div id="faqs">
  ## Preguntas frecuentes
</div>

<AccordionGroup>
  <Accordion title="¿Cómo mejora Firecrawl los procesos de enriquecimiento de leads?">
    Firecrawl extrae automáticamente información de la empresa, datos de contacto, cartera de productos y noticias recientes de los sitios web de prospectos. Esto enriquece tu CRM con información precisa y actualizada para optimizar la prospección y acercamiento de ventas.
  </Accordion>

  <Accordion title="¿Puede Firecrawl encontrar información de contacto en sitios web?">
    ¡Sí! Firecrawl extrae información de contacto de acceso público, incluidos correos electrónicos y números de teléfono, desde sitios web de empresas, páginas de equipo y secciones de contacto.
  </Accordion>

  <Accordion title="¿Qué tan precisos son los datos de leads de Firecrawl?">
    Como Firecrawl extrae datos directamente de sitios web en producción, obtienes la información más actual disponible. Esto es más preciso que las bases de datos estáticas, que se desactualizan rápidamente.
  </Accordion>

  <Accordion title="¿Puedo integrar Firecrawl con mi CRM?">
    Sí. Usa nuestra API o la integración con Zapier para enriquecer automáticamente leads en Salesforce, HubSpot, Pipedrive y otros CRM. Mantén tus datos de leads actualizados sin investigación manual.
  </Accordion>

  <Accordion title="¿Cómo ayuda Firecrawl con el marketing basado en cuentas?">
    Extrae información detallada de la empresa, actualizaciones recientes y eventos detonantes de los sitios web de cuentas objetivo. Esta inteligencia ayuda a personalizar el acercamiento e identificar el momento ideal para interactuar.
  </Accordion>
</AccordionGroup>
