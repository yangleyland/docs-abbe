Source de référence Java canonique de Firecrawl pour les agents. Générée à partir du code source du SDK et de la spécification OpenAPI v2.

<div id="install">
  ## Installation
</div>

Maven :

```xml
<dependency>
  <groupId>com.firecrawl</groupId>
  <artifactId>firecrawl-java</artifactId>
  <version>1.2.0</version>
</dependency>
```

Gradle :

```gradle
implementation("com.firecrawl:firecrawl-java:1.2.0")
```

<div id="authenticate">
  ## Authentification
</div>

```java
import com.firecrawl.client.FirecrawlClient;

FirecrawlClient client = FirecrawlClient.builder()
    .apiKey(System.getenv("FIRECRAWL_API_KEY"))
    .build();
```

<div id="when-to-use-what">
  ## Quand utiliser quoi
</div>

* `search` : à utiliser si vous partez d&#39;une requête et souhaitez découvrir des résultats.
* `scrape` : à utiliser si vous avez déjà une URL et voulez récupérer le contenu de la page.
* `interact` : à utiliser si la page nécessite des clics, des formulaires ou des actions du navigateur après le scraping.
* `support/ask` : à utiliser lorsqu’un appel à l’API Firecrawl échoue ou renvoie des résultats inattendus et que vous avez besoin d’un diagnostic.
* `support/docs-search` : à utiliser lorsque vous avez besoin de consulter la documentation de Firecrawl.

<div id="search">
  ## Recherche
</div>

### Pourquoi l’utiliser

Utilisez la recherche pour découvrir des pages pertinentes à partir d’une requête, puis choisir des URL à scraper ou avec lesquelles interagir. Vous pouvez limiter les résultats à un site avec `site:`, par exemple `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-methods">
  ### Méthodes recommandées du SDK
</div>

* `client.search(query)`
* `client.search(query, options)`

<div id="return-value">
  ### Valeur de retour
</div>

`search` renvoie `SearchData`. Accédez aux groupes de résultats avec `getWeb()`, `getNews()` et `getImages()` (chacun est de type `List<Map<String, Object>>` et peut être nul). Ne traitez pas `SearchData` comme une liste de résultats directement itérable.

<div id="simple-example">
  ### Exemple simple
</div>

```java
import com.firecrawl.models.SearchData;
import java.util.List;
import java.util.Map;

SearchData results = client.search("site:docs.firecrawl.dev webhook retries");
List<Map<String, Object>> web = results.getWeb();
```

<div id="complex-example">
  ### Exemple complexe
</div>

```java
import com.firecrawl.models.SearchOptions;
import com.firecrawl.models.ScrapeOptions;
import com.firecrawl.models.JsonFormat;
import com.firecrawl.models.LocationConfig;

SearchOptions options = SearchOptions.builder()
    .sources(List.of("web", "news"))
    .categories(List.of("research"))
    .limit(10)
    .tbs("qdr:m")
    .location("San Francisco,California,United States")
    .ignoreInvalidURLs(true)
    .timeout(60000)
    .scrapeOptions(
        ScrapeOptions.builder()
            .formats(List.of(
                "markdown",
                "links",
                JsonFormat.builder().prompt("Extract title and key endpoints.").build()
            ))
            .onlyMainContent(true)
            .waitFor(1000)
            .build()
    )
    .build();

SearchData results = client.search("site:docs.firecrawl.dev crawl webhooks", options);
```

<div id="parameters">
  ### Paramètres
</div>

* `query`
  * Type : String
  * À utiliser lorsque : vous avez besoin d’une requête de recherche.
  * Remarques : utilisez `site:example.com` pour limiter les résultats à un domaine.

* `options.sources`
  * Type : liste de chaînes de caractères de source ou de cartographies
  * À utiliser lorsque : vous souhaitez contrôler les sources à interroger.
  * Valeurs confirmées :
    * `"web"` : résultats de l’index web
    * `"news"` : résultats d’actualités
    * `"images"` : résultats d’images
    * `{type: "web" | "news" | "images"}` : forme de cartographie de source typée

* `options.categories`
  * Type : liste de chaînes de caractères de catégorie ou de cartographies
  * À utiliser lorsque : vous souhaitez filtrer les résultats par catégorie.
  * Valeurs confirmées :
    * `"github"` : résultats axés sur GitHub
    * `"research"` : résultats de recherche et académiques
    * `"pdf"` : résultats axés sur les PDF
    * `{type: "github" | "research" | "pdf"}` : forme de cartographie de catégorie typée

* `options.limit`
  * Type : Integer
  * À utiliser lorsque : vous souhaitez limiter le nombre de résultats.

* `options.tbs`
  * Type : String
  * À utiliser lorsque : vous avez besoin d’un filtre temporel (par exemple `qdr:d`, `qdr:w`, `sbd:1,qdr:m`).

* `options.location`
  * Type : String
  * À utiliser lorsque : vous souhaitez obtenir des résultats localisés.

* `options.ignoreInvalidURLs`
  * Type : Boolean
  * À utiliser lorsque : vous souhaitez exclure les URL qui ne peuvent pas être récupérées par d’autres points de terminaison.

* `options.timeout`
  * Type : Integer
  * À utiliser lorsque : vous avez besoin d’un délai d’expiration de requête en millisecondes.

* `options.scrapeOptions`
  * Type : `ScrapeOptions`
  * À utiliser lorsque : vous souhaitez extraire chaque résultat de recherche (voir les paramètres de Scrape pour les champs).

* `options.integration`
  * Type : String
  * À utiliser lorsque : l’API attend un identifiant d’intégration dans la requête.

<div id="scrape">
  ## Scrape
</div>

<div id="why-use-it">
  ### Pourquoi l&#39;utiliser
</div>

Utilisez scrape si vous avez déjà une URL et souhaitez du contenu structuré dans un ou plusieurs formats.

<div id="preferred-sdk-methods">
  ### Méthodes recommandées du SDK
</div>

* `client.scrape(url)`
* `client.scrape(url, options)`

<div id="return-value">
  ### Valeur de retour
</div>

`scrape` renvoie un `Document`. Les getters les plus courants incluent `getMarkdown()`, `getHtml()`, `getRawHtml()`, `getJson()`, `getMetadata()`, `getLinks()`, ainsi que des champs supplémentaires lorsque les formats correspondants sont demandés.

<div id="simple-example">
  ### Exemple simple
</div>

```java
Document doc = client.scrape(
    "https://docs.firecrawl.dev",
    ScrapeOptions.builder().formats(List.of("markdown")).build()
);
```

<div id="complex-example">
  ### Exemple complexe
</div>

```java
import com.firecrawl.models.ScrapeOptions;
import com.firecrawl.models.JsonFormat;

List<Map<String, Object>> actions = List.of(
    Map.of("type", "click", "selector", "#accept"),
    Map.of("type", "wait", "milliseconds", 750),
    Map.of("type", "scrape")
);

LocationConfig location = LocationConfig.builder()
    .country("US")
    .languages(List.of("en-US"))
    .build();

ScrapeOptions options = ScrapeOptions.builder()
    .formats(List.of(
        "markdown",
        "links",
        JsonFormat.builder().prompt("Extract plan names and prices.").build(),
        Map.of("type", "screenshot", "fullPage", true, "quality", 80)
    ))
    .onlyMainContent(true)
    .waitFor(1000)
    .parsers(List.of(Map.of("type", "pdf", "maxPages", 5)))
    .actions(actions)
    .location(location)
    .removeBase64Images(true)
    .blockAds(true)
    .proxy("auto")
    .maxAge(86400000L)
    .storeInCache(true)
    .build();

Document doc = client.scrape("https://example.com/pricing", options);
```

<div id="parameters">
  ### Paramètres
</div>

* `url`
  * Type : String
  * À utiliser lorsque : vous voulez scraper une page précise.

* `options.formats`
  * Type : liste de chaînes de format ou de cartographies de format
  * À utiliser lorsque : vous voulez plusieurs formats de sortie.
  * Chaînes de format confirmées :
    * `"markdown"` : contenu markdown
    * `"html"` : HTML nettoyé
    * `"rawHtml"` : HTML brut
    * `"links"` : liens de la page
    * `"images"` : URL d&#39;images
    * `"screenshot"` : sortie de capture d&#39;écran
    * `"summary"` : sortie de résumé
    * `"changeTracking"` : sortie de suivi des modifications
    * `"json"` : extraction JSON
    * `"attributes"` : extraction d&#39;attributs
    * `"branding"` : sortie du profil de branding
    * `"audio"` : extraction audio
    * `"video"` : extraction vidéo
  * Champs de l&#39;objet de format :
    * `type` : l&#39;une des chaînes de format ci-dessus
    * `prompt`, `schema` : options d&#39;extraction JSON pour `type: "json"`
    * `modes`, `schema`, `prompt`, `tag` : options de suivi des modifications pour `type: "changeTracking"`
    * `fullPage`, `quality`, `viewport` : options de capture d&#39;écran pour `type: "screenshot"`
    * `selectors` : tableau de `{selector, attribute}` pour `type: "attributes"`

* `options.headers`
  * Type : `Map<String, String>`
  * À utiliser lorsque : vous avez besoin d&#39;en-têtes de requête personnalisés.

* `options.includeTags`
  * Type : `List<String>`
  * À utiliser lorsque : vous voulez inclure uniquement certaines balises HTML.

* `options.excludeTags`
  * Type : `List<String>`
  * À utiliser lorsque : vous voulez exclure certaines balises HTML.

* `options.onlyMainContent`
  * Type : Boolean
  * À utiliser lorsque : vous voulez supprimer la navigation, le pied de page et les autres éléments récurrents.

* `options.timeout`
  * Type : Integer
  * À utiliser lorsque : vous avez besoin d&#39;un délai d&#39;expiration en millisecondes.

* `options.waitFor`
  * Type : Integer
  * À utiliser lorsque : vous devez attendre le rendu de la page (en millisecondes).

* `options.mobile`
  * Type : Boolean
  * À utiliser lorsque : vous voulez un viewport mobile.

* `options.parsers`
  * Type : `List<Object>`
  * À utiliser lorsque : vous avez besoin de contrôles de parsing de fichiers.
  * Valeurs confirmées :
    * `"pdf"`
    * `{type: "pdf", maxPages: number}`

* `options.actions`
  * Type : `List<Map<String, Object>>`
  * À utiliser lorsque : vous avez besoin d&#39;actions légères avant le scraping.
  * Types d&#39;action confirmés :
    * `wait` : `milliseconds` ou `selector` requis
    * `screenshot` : `fullPage`, `quality`, `viewport` facultatifs
    * `click` : `selector` requis
    * `write` : `text` requis (cliquez d&#39;abord pour donner le focus au champ)
    * `press` : `key` requis
    * `scroll` : `direction` (`up` ou `down`) requis, `selector` facultatif
    * `scrape` : aucun champ supplémentaire
    * `executeJavascript` : `script` requis
    * `pdf` : `format` (A0, A1, A2, A3, A4, A5, A6, Letter, Legal, Tabloid, Ledger), `landscape`, `scale` facultatifs

* `options.location`
  * Type : `LocationConfig`
  * À utiliser lorsque : vous avez besoin d&#39;un scraping tenant compte de la géolocalisation ou de la langue.

* `options.skipTlsVerification`
  * Type : Boolean
  * À utiliser lorsque : vous devez ignorer la vérification TLS.

* `options.removeBase64Images`
  * Type : Boolean
  * À utiliser lorsque : vous voulez supprimer les images en base64 de la sortie markdown.

* `options.blockAds`
  * Type : Boolean
  * À utiliser lorsque : vous voulez bloquer les publicités et les pop-ups de cookies.

* `options.proxy`
  * Type : String
  * À utiliser lorsque : vous avez besoin de contrôler le proxy.
  * Valeurs confirmées : `"basic"`, `"stealth"`, `"enhanced"`, `"auto"` ou une chaîne d&#39;URL de proxy personnalisée

* `options.maxAge`
  * Type : Long
  * À utiliser lorsque : vous voulez des données mises en cache jusqu&#39;à un âge maximal (en millisecondes).

* `options.storeInCache`
  * Type : Boolean
  * À utiliser lorsque : vous voulez que Firecrawl mette le résultat en cache.

* `options.integration`
  * Type : String
  * À utiliser lorsque : l&#39;API attend un identifiant d&#39;intégration dans la requête.

<div id="interact">
  ## Interact
</div>

### Pourquoi l’utiliser

Utilisez interact lorsqu’une page nécessite des actions du navigateur ou l’exécution de code une fois le scrape lancé.

<div id="preferred-sdk-methods">
  ### Méthodes recommandées du SDK
</div>

* `client.interact(jobId, code)` — utilise le langage `node` par défaut et le délai d’exécution par défaut de l’API
* `client.interact(jobId, code, language, timeout)` — `timeout` est exprimé en secondes (1–300), ou `null` pour l’omettre et utiliser la valeur par défaut de l’API (30 secondes)
* `client.interact(jobId, code, language, timeout, origin)` — la chaîne `origin`, facultative, n’est envoyée que si elle n’est pas nulle (attribution de la requête)

<div id="simple-example">
  ### Exemple simple
</div>

```java
import com.firecrawl.models.BrowserExecuteResponse;

BrowserExecuteResponse result = client.interact(
    "<scrapeJobId>",
    "console.log(await page.title());",
    "node",
    60
);
```

<div id="complex-example">
  ### Exemple complexe
</div>

```java
BrowserExecuteResponse result = client.interact(
    "<scrapeJobId>",
    "// Utilisez les méthodes de page Playwright ici",
    "node",
    120
);
```

<div id="stop-interactive-browser">
  ### Arrêter le navigateur interactif
</div>

Mettez fin à la session de navigateur associée au scrape une fois l’opération terminée.

**Méthode recommandée du SDK :** `client.stopInteractiveBrowser(jobId)`

```java
import com.firecrawl.models.BrowserDeleteResponse;

BrowserDeleteResponse stopped = client.stopInteractiveBrowser("<scrapeJobId>");
```

<div id="interact-response-browserexecuteresponse">
  ### Réponse d’Interact (`BrowserExecuteResponse`)
</div>

* `isSuccess()`: booléen
* `getStdout()`, `getStderr()`, `getResult()`, `getError()`: String (peut être null)
* `getExitCode()`: Integer (peut être null)
* `getKilled()`: Boolean (peut être null) — true lorsque l’exécution a été arrêtée à cause d’un timeout

<div id="stop-response-browserdeleteresponse">
  ### Réponse d’arrêt (`BrowserDeleteResponse`)
</div>

* `isSuccess()`: boolean
* `getSessionDurationMs()`: Long (peut être `null`)
* `getCreditsBilled()`: Integer (peut être `null`)
* `getError()`: String (peut être `null`)

<div id="parameters">
  ### Paramètres
</div>

* `jobId`
  * Type : String
  * À utiliser lorsque : vous avez un ID de tâche de scraping.

* `code`
  * Type : String
  * À utiliser lorsque : vous souhaitez exécuter du code dans la session de navigateur.

* `language`
  * Type : String
  * À utiliser lorsque : vous avez besoin d’un environnement d’exécution spécifique.
  * Valeurs confirmées : `"python"`, `"node"`, `"bash"`
  * Remarques : dans le SDK, la valeur par défaut est `"node"` si la valeur est nulle.

* `timeout`
  * Type : Integer
  * À utiliser lorsque : vous avez besoin d’un délai d’expiration d’exécution en secondes (1–300). Une valeur nulle omet le champ et utilise la valeur par défaut de l’API.

* `origin`
  * Type : String
  * À utiliser lorsque : vous avez besoin d’un libellé `origin` facultatif sur la requête. Préférez l’omettre, sauf si votre intégration l’exige.

<div id="notes">
  ## Remarques
</div>

* Alias dépréciés : `scrapeExecute` → `interact`, `deleteScrapeBrowser` → `stopInteractiveBrowser` (ainsi que les helpers `*Async` correspondants).
* Le Java SDK n&#39;expose que des interactions pilotées par le code : il n&#39;y a pas de paramètre `prompt` sur `interact` (contrairement à certains SDK dans d&#39;autres langages).
