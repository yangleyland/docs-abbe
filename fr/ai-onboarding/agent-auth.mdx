Si vous avez besoin d’une connexion dans le navigateur ou d’une clé API du dashboard, utilisez l’onboarding avec intervention humaine via [`SKILL.md`](https://www.firecrawl.dev/agent-onboarding/SKILL.md). Consultez [Obtenir les identifiants](/fr/ai-onboarding#get-credentials).

Cette page résume l’enregistrement WorkOS ID-JAG pour Firecrawl. Utilisez [`auth.md`](https://www.firecrawl.dev/auth.md) pour les instructions complètes d’enregistrement, les exigences relatives aux claims et la gestion des erreurs.

<div id="using-workos-id-jag">
  ## Utiliser WorkOS ID-JAG
</div>

Utilisez cette option lorsque votre plateforme d’agents peut générer un WorkOS ID-JAG pour Firecrawl.

<Card title="Ouvrir auth.md" icon="file-lines" href="https://www.firecrawl.dev/auth.md">
  Instructions d’enregistrement des agents sensibles à l’identité. Suivez les étapes dans l’ordre.
</Card>

Pour les exigences relatives aux claims, à la découverte, à l’enregistrement, aux limites et aux erreurs, utilisez [`auth.md`](https://www.firecrawl.dev/auth.md).
