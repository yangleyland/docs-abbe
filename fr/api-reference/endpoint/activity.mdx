Affiche votre activité API des dernières 24 heures. Utilisez ceci pour repérer les ID de tâche, puis obtenez les résultats via le point de terminaison GET correspondant.

| Point de terminaison | Point de terminaison de récupération |
| -------------------- | ------------------------------------ |
| `scrape`             | `GET /v2/scrape/{id}`                |
| `crawl`              | `GET /v2/crawl/{id}`                 |
| `batch_scrape`       | `GET /v2/batch/scrape/{id}`          |
| `agent`              | `GET /v2/extract/{id}`               |
