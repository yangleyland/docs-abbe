Firecrawl est open source sous la [licence AGPL-3.0](https://github.com/firecrawl/firecrawl/blob/main/LICENSE). Pour offrir le meilleur produit possible, nous proposons également une version cloud hébergée sur [firecrawl.dev](https://firecrawl.dev) avec des fonctionnalités supplémentaires et une infrastructure managée.

<div id="feature-comparison">
  ## Comparaison des fonctionnalités
</div>

La version open source inclut le moteur principal de scraping et de crawling. Firecrawl Cloud ajoute des capacités d&#39;infrastructure, un tableau de bord d&#39;administration et des fonctionnalités entreprise, en plus de tout ce que propose l&#39;édition open source.

| Scrape | ✔ | ✔ |

| Map | ✔ | ✔ |
| Search | ✔ | ✔ |

| mode JSON | ✔ | ✔ |
| formats prêts pour les LLM | ✔ | ✔ |

| Rotations de proxy | ✘ | ✔ |
| Tableau de bord | ✘ | ✔ |
| Fonctionnalités entreprise | ✘ | ✔ |

<div id="why-both">
  ## Pourquoi les deux ?
</div>

La solution cloud nous permet d’innover en continu et de maintenir un service de haute qualité, durable, pour tous les utilisateurs. Les revenus de Firecrawl Cloud financent le développement continu du projet open source.

* **Open source** — Exécutez Firecrawl sur votre propre infrastructure avec un contrôle total sur le moteur de scraping principal. Voir [Exécution en local](/fr/contributing/guide) et [Auto‑hébergement](/fr/contributing/self-host).
* **Cloud** — Commencez immédiatement avec un service géré incluant des proxys améliorés, un tableau de bord d’utilisation et des fonctionnalités pour les entreprises. Inscrivez-vous sur [firecrawl.dev](https://firecrawl.dev).
