Ajoutez à Claude.ai des fonctionnalités de scraping web et de recherche avec Firecrawl MCP grâce à des connecteurs personnalisés.

<Note>
  Vous cherchez comment configurer Claude Code ? Consultez plutôt le [guide
  Claude Code](/fr/quickstarts/claude-code).
</Note>

<div id="quick-setup">
  ## Configuration rapide
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenez votre clé API
</div>

Créez un compte sur [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys), puis copiez votre clé API.

<div id="2-add-custom-connector">
  ### 2. Ajouter un connecteur personnalisé
</div>

Accédez à [Settings &gt; Connectors](https://claude.ai/settings/connectors) dans Claude.ai, puis cliquez sur **Add custom connector**.

Renseignez les informations du connecteur :

* **URL :** `https://mcp.firecrawl.dev/YOUR_API_KEY/v2/mcp`
* **OAuth Client ID :** laissez vide
* **OAuth Client Secret :** laissez vide

Remplacez `YOUR_API_KEY` dans l’URL par votre [clé API Firecrawl](https://www.firecrawl.dev/app/api-keys). Votre clé API étant intégrée directement dans l’URL, aucun champ d’authentification supplémentaire n’est nécessaire.

Cliquez sur **Add** pour enregistrer le connecteur.

<Tip>
  **Vous préférez ne pas mettre votre clé API dans l’URL ?** Utilisez plutôt le point de terminaison sans clé `https://mcp.firecrawl.dev/v2/mcp` (en laissant toujours les champs OAuth vides). Claude.ai ouvrira une fenêtre de navigateur pour vous permettre de vous connecter à Firecrawl et d’autoriser le connecteur. Consultez [Connect MCP Clients with OAuth](/fr/developer-guides/mcp-setup-guides/oauth) pour plus de détails.
</Tip>

<div id="3-enable-in-conversation">
  ### 3. Activer dans la conversation
</div>

Dans n’importe quelle conversation sur Claude.ai, cliquez sur le bouton **+** en bas à gauche, ouvrez **Connectors**, puis activez le connecteur Firecrawl.

<div id="quick-demo">
  ## Démo rapide
</div>

Avec le connecteur Firecrawl activé, essayez les prompts suivants :

**Rechercher sur le web :**

```
Search for the latest Next.js 15 features
```

**Scraper une page :**

```
Scrape firecrawl.dev and tell me what it does
```

**Obtenir la documentation :**

```
Find and scrape the Stripe API docs for payment intents
```

Claude utilisera automatiquement les outils de recherche et de scraping de Firecrawl pour obtenir les informations.
