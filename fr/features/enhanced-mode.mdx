Firecrawl propose différents types de proxy pour vous aider à effectuer du scraping sur des sites web de complexité variable. Définissez le paramètre `proxy` pour choisir la stratégie de proxy utilisée pour une requête.


<div id="proxy-types">
  ## Types de proxy
</div>

Firecrawl prend en charge trois types de proxy :

| Type | Description | Vitesse | Coût |
|------|-------------|---------|------|
| `basic` | Proxies standard adaptés à la plupart des sites | Rapide | 1 crédit |
| `enhanced` | Proxies avancés pour les sites complexes | Plus lent | 5 crédits par requête |
| `auto` | Essaie d’abord `basic`, puis réessaie avec `enhanced` en cas d’échec | Variable | 1 crédit si `basic` réussit, 5 crédits si `enhanced` est nécessaire |

Si vous ne spécifiez pas de proxy, Firecrawl utilise `auto` par défaut.
