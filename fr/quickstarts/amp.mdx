Ajoutez à [Sourcegraph Amp](https://ampcode.com), via MCP, les outils de recherche, de scrape, de crawl et de navigateur de Firecrawl.

<div id="quick-setup">
  ## Prise en main rapide
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenez votre clé API
</div>

Inscrivez-vous sur [firecrawl.dev/app](https://www.firecrawl.dev/app) et copiez votre clé API.

<div id="2-add-firecrawl-to-amp">
  ### 2. Ajouter Firecrawl à Amp
</div>

Ouvrez les paramètres d’Amp et ajoutez un serveur MCP. Amp accepte une configuration MCP standard :

```json
{
  "amp.mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Remplacez `fc-YOUR-API-KEY` par votre clé API Firecrawl.

<div id="3-reload-amp">
  ### 3. Actualiser Amp
</div>

Actualisez la fenêtre Amp. Les outils Firecrawl sont désormais disponibles pour l’agent.

<div id="quick-demo">
  ## Démo rapide
</div>

```
Search the web for "Sourcegraph Cody vs Amp" and summarize the differences.
```

```
Scrape https://docs.firecrawl.dev and list the core endpoints.
```

```
Crawl https://example.com and output a site map as JSON.
```

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hébergée sur un serveur distant (Node.js non requis)
</div>

```json
{
  "amp.mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Dépannage
</div>

* **Le serveur ne parvient pas à démarrer** — consultez la vue des logs MCP d&#39;Amp pour voir la sortie stderr.
* **`npx` manquant** — installez Node.js 18+ ou utilisez l&#39;URL hébergée à distance indiquée ci-dessus.
