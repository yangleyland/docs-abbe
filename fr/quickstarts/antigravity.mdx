Ajoutez à [Google Antigravity](https://antigravity.google/) les outils Firecrawl de recherche, de scraping, de crawl et de navigateur via MCP.

<div id="quick-setup">
  ## Configuration rapide
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenez votre clé API
</div>

Inscrivez-vous sur [firecrawl.dev/app](https://www.firecrawl.dev/app), puis copiez votre clé API.

<div id="2-add-firecrawl-to-antigravity">
  ### 2. Ajouter Firecrawl à Antigravity
</div>

Ouvrez les paramètres d’Antigravity (`Cmd/Ctrl + ,`), recherchez **MCP Servers**, puis ajoutez un nouveau serveur avec :

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Remplacez `fc-YOUR-API-KEY` par votre clé API Firecrawl.

<div id="3-reload-antigravity">
  ### 3. Recharger Antigravity
</div>

Rechargez la fenêtre (`Cmd/Ctrl + Shift + P` → `Reload Window`). L’agent a désormais accès aux outils web de Firecrawl.

<div id="quick-demo">
  ## Démo rapide
</div>

Dans une conversation avec un agent Antigravity :

```
Search the web for "Vercel AI SDK v5 release notes" and summarize.
```

```
Scrape https://docs.firecrawl.dev/ai-onboarding and list every linked guide.
```

```
Crawl https://example.com and extract every page title.
```

Antigravity achemine automatiquement ces appels d’outils via Firecrawl MCP.

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hébergée à distance (sans Node.js)
</div>

```json
{
  "mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Dépannage
</div>

* **Le serveur apparaît comme « failed »** — consultez le panneau de sortie MCP pour `stderr`. La plupart des échecs sont dus à une clé API manquante ou à l’absence de `npx` dans le `PATH`.
* **Outils non appelés** — mentionnez explicitement Firecrawl (par exemple, « Utilisez Firecrawl pour faire du scraping… ») dans votre premier prompt afin que l’agent choisisse le bon outil.
