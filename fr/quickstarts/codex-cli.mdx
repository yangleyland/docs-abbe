Ajoutez à [OpenAI Codex CLI](https://github.com/openai/codex), via MCP, les outils de recherche, de scraping, de crawl et de browser de Firecrawl.

<div id="quick-setup">
  ## Configuration rapide
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenez votre clé API
</div>

Inscrivez-vous sur [firecrawl.dev/app](https://www.firecrawl.dev/app) et copiez votre clé API.

<div id="2-add-firecrawl-to-codex">
  ### 2. Ajouter Firecrawl à Codex
</div>

Codex lit la configuration du serveur MCP dans `~/.codex/config.toml`. Ajoutez le serveur Firecrawl :

```toml
[mcp_servers.firecrawl]
command = "npx"
args = ["-y", "firecrawl-mcp"]

[mcp_servers.firecrawl.env]
FIRECRAWL_API_KEY = "fc-YOUR-API-KEY"
```

Remplacez `fc-YOUR-API-KEY` par votre clé API Firecrawl.

<div id="3-start-codex">
  ### 3. Démarrer Codex
</div>

```bash
codex
```

Codex détecte les outils Firecrawl au lancement. Vérifiez qu’ils sont bien chargés :

```bash
/mcp
```

Vous devriez voir `firecrawl` apparaître dans la liste, avec des outils tels que `firecrawl_search`, `firecrawl_scrape`, `firecrawl_crawl` et `firecrawl_extract`.

<div id="quick-demo">
  ## Démo rapide
</div>

Essayez les prompts suivants :

```
Search the web for the latest Next.js App Router release notes and summarize.
```

```
Scrape https://docs.firecrawl.dev and list the top-level sections.
```

```
Crawl https://example.com and save the markdown for every page under /blog.
```

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hébergée à distance (sans Node.js)
</div>

Si vous préférez ne pas exécuter `npx` en local :

```toml
[mcp_servers.firecrawl]
url = "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
```

<div id="troubleshooting">
  ## Dépannage
</div>

* **Codex ne voit pas les outils** — exécutez `codex --version` pour vérifier que vous utilisez une version compatible MCP, puis redémarrez le CLI après avoir modifié `config.toml`.
* **`spawn npx ENOENT`** — installez Node.js 18+ et assurez-vous que `npx` est dans votre `PATH`, ou utilisez l’URL hébergée à distance ci-dessus.
* **401 / clé invalide** — régénérez une clé API sur [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys).
