Ajoutez à [Google Gemini CLI](https://github.com/google-gemini/gemini-cli) les outils Firecrawl de recherche, de scraping, de crawl et de navigation via MCP.

<div id="quick-setup">
  ## Configuration rapide
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenez votre clé API
</div>

Inscrivez-vous sur [firecrawl.dev/app](https://www.firecrawl.dev/app) et copiez votre clé API.

<div id="2-add-firecrawl-to-gemini-cli">
  ### 2. Ajouter Firecrawl à Gemini CLI
</div>

Gemini CLI lit la configuration MCP à partir de `~/.gemini/settings.json` (global) ou de `.gemini/settings.json` dans votre projet. Ajoutez :

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Remplacez `fc-YOUR-API-KEY` par votre clé API Firecrawl.

<div id="3-launch-gemini-cli">
  ### 3. Lancez Gemini CLI
</div>

```bash
gemini
```

Vérifiez que le serveur est bien lancé :

```
/mcp list
```

Vous devriez voir `firecrawl` et ses outils.

<div id="quick-demo">
  ## Démo rapide
</div>

```
Use firecrawl to search the web for "Gemini 2.5 context window" and summarize the top 5 results.
```

```
Scrape https://ai.google.dev/gemini-api/docs and outline the sections.
```

```
Crawl https://example.com and extract the product names from /products.
```

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hébergée à distance (sans Node.js)
</div>

```json
{
  "mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Dépannage
</div>

* **Les outils n’apparaissent pas** — redémarrez Gemini CLI après avoir modifié `settings.json` ; les serveurs MCP sont chargés au démarrage.
* **`spawn npx ENOENT`** — installez Node.js 18+ ou utilisez l’URL hébergée à distance.
* **Limite de débit atteinte** — passez à une offre Firecrawl supérieure sur [firecrawl.dev/pricing](https://www.firecrawl.dev/pricing).
