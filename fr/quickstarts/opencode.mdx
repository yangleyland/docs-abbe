Ajoutez à [OpenCode](https://opencode.ai), via MCP, les outils de recherche, de scraping, de crawl et de navigateur de Firecrawl.

<div id="quick-setup">
  ## Prise en main rapide
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenez votre clé API
</div>

Inscrivez-vous sur [firecrawl.dev/app](https://www.firecrawl.dev/app) et copiez votre clé API.

<div id="2-add-firecrawl-to-opencode">
  ### 2. Ajouter Firecrawl à OpenCode
</div>

OpenCode lit sa configuration depuis `~/.config/opencode/config.json` (globale) ou `./opencode.json` dans votre projet. Ajoutez :

```json
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "firecrawl": {
      "type": "local",
      "command": ["npx", "-y", "firecrawl-mcp"],
      "environment": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Remplacez `fc-YOUR-API-KEY` par votre clé API Firecrawl.

<div id="3-start-opencode">
  ### 3. Démarrer OpenCode
</div>

```bash
opencode
```

OpenCode charge les serveurs MCP au démarrage. Vérifiez que Firecrawl est bien connecté :

```
/mcp
```

<div id="quick-demo">
  ## Démo rapide
</div>

```
Search the web for "Bun 2.0 changelog" and summarize the top results.
```

```
Scrape https://docs.firecrawl.dev/introduction and list the code examples.
```

```
Crawl https://example.com/blog and save each post as markdown.
```

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hébergée à distance (sans Node.js)
</div>

```json
{
  "mcp": {
    "firecrawl": {
      "type": "remote",
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Dépannage
</div>

* **Serveur non connecté** — exécutez `opencode doctor` pour examiner les erreurs de chargement du MCP.
* **Permission refusée avec `npx`** — installez Node.js 18+ et assurez-vous que votre shell prend bien en compte l&#39;installation (`which npx`).
