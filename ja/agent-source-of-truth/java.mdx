エージェント向け Firecrawl Java の標準リファレンスです。SDK のソースコードと v2 OpenAPI 仕様から生成されています。

<div id="install">
  ## インストール
</div>

Maven:

```xml
<dependency>
  <groupId>com.firecrawl</groupId>
  <artifactId>firecrawl-java</artifactId>
  <version>1.2.0</version>
</dependency>
```

Gradle:

```gradle
implementation("com.firecrawl:firecrawl-java:1.2.0")
```

<div id="authenticate">
  ## 認証
</div>

```java
import com.firecrawl.client.FirecrawlClient;

FirecrawlClient client = FirecrawlClient.builder()
    .apiKey(System.getenv("FIRECRAWL_API_KEY"))
    .build();
```

<div id="when-to-use-what">
  ## 用途別の使い分け
</div>

* `search`: クエリから始めて、まず対象を見つけたい場合に使います。
* `scrape`: すでに URL があり、ページのコンテンツを取得したい場合に使います。
* `interact`: ページ上でクリックやフォーム入力、またはスクレイピング後のブラウザアクションが必要な場合に使います。
* `support/ask`: Firecrawl API の呼び出しが失敗したり、想定外の結果が返されたりしたときに、原因を診断する必要がある場合に使います。
* `support/docs-search`: Firecrawl のドキュメントを調べたい場合に使います。

<div id="search">
  ## 検索
</div>

### これを使う場面

クエリから関連ページを見つけ、スクレイピングまたは Interact する URL を選びたい場合は、search を使います。たとえば、`site:` を使うと、結果を特定のサイトに絞り込めます。例: `site:docs.firecrawl.dev crawl webhooks`

<div id="preferred-sdk-methods">
  ### 推奨SDKメソッド
</div>

* `client.search(query)`
* `client.search(query, options)`

<div id="return-value">
  ### 戻り値
</div>

`search` は `SearchData` を返します。結果の各バケットは `getWeb()`、`getNews()`、`getImages()` で取得します (いずれも `List<Map<String, Object>>` で、null の場合があります) 。`SearchData` をヒットのリストとして直接反復処理できるものとして扱わないでください。

<div id="simple-example">
  ### シンプルな例
</div>

```java
import com.firecrawl.models.SearchData;
import java.util.List;
import java.util.Map;

SearchData results = client.search("site:docs.firecrawl.dev webhook retries");
List<Map<String, Object>> web = results.getWeb();
```

<div id="complex-example">
  ### 複雑な例
</div>

```java
import com.firecrawl.models.SearchOptions;
import com.firecrawl.models.ScrapeOptions;
import com.firecrawl.models.JsonFormat;
import com.firecrawl.models.LocationConfig;

SearchOptions options = SearchOptions.builder()
    .sources(List.of("web", "news"))
    .categories(List.of("research"))
    .limit(10)
    .tbs("qdr:m")
    .location("San Francisco,California,United States")
    .ignoreInvalidURLs(true)
    .timeout(60000)
    .scrapeOptions(
        ScrapeOptions.builder()
            .formats(List.of(
                "markdown",
                "links",
                JsonFormat.builder().prompt("Extract title and key endpoints.").build()
            ))
            .onlyMainContent(true)
            .waitFor(1000)
            .build()
    )
    .build();

SearchData results = client.search("site:docs.firecrawl.dev crawl webhooks", options);
```

<div id="parameters">
  ### パラメータ
</div>

* `query`
  * Type: String
  * 使用する場面: 検索クエリが必要な場合。
  * 注記: 結果を特定のドメインに絞り込むには `site:example.com` を使用します。

* `options.sources`
  * Type: List of source strings or maps
  * 使用する場面: 検索対象のソースを制御したい場合。
  * 確認済みの値:
    * `"web"`: Web インデックスの結果
    * `"news"`: ニュース結果
    * `"images"`: 画像結果
    * `{type: "web" | "news" | "images"}`: 型付きソースマップ形式

* `options.categories`
  * Type: List of category strings or maps
  * 使用する場面: 結果をカテゴリで絞り込みたい場合。
  * 確認済みの値:
    * `"github"`: GitHub に特化した結果
    * `"research"`: 研究・学術関連の結果
    * `"pdf"`: PDF に特化した結果
    * `{type: "github" | "research" | "pdf"}`: 型付きカテゴリマップ形式

* `options.limit`
  * Type: Integer
  * 使用する場面: 結果数の上限を設定したい場合。

* `options.tbs`
  * Type: String
  * 使用する場面: 時間ベースの絞り込みが必要な場合 (例: `qdr:d`, `qdr:w`, `sbd:1,qdr:m`) 。

* `options.location`
  * Type: String
  * 使用する場面: ロケーションに応じた結果を取得したい場合。

* `options.ignoreInvalidURLs`
  * Type: Boolean
  * 使用する場面: 他のエンドポイントでスクレイピングできない URL を除外したい場合。

* `options.timeout`
  * Type: Integer
  * 使用する場面: リクエストのタイムアウトをミリ秒単位で指定する必要がある場合。

* `options.scrapeOptions`
  * Type: `ScrapeOptions`
  * 使用する場面: 各検索結果をスクレイピングしたい場合 (フィールドについては Scrape パラメータを参照) 。

* `options.integration`
  * Type: String
  * 使用する場面: API がリクエストにインテグレーション識別子を要求する場合。

<div id="scrape">
  ## スクレイピング
</div>

<div id="why-use-it">
  ### どのような場合に使うか
</div>

URL がすでにわかっていて、1 つ以上のフォーマットで構造化されたコンテンツを取得したい場合は、scrape を使用します。

<div id="preferred-sdk-methods">
  ### 推奨SDKメソッド
</div>

* `client.scrape(url)`
* `client.scrape(url, options)`

<div id="return-value">
  ### 戻り値
</div>

`scrape` は `Document` を返します。一般的な getter には `getMarkdown()`, `getHtml()`, `getRawHtml()`, `getJson()`, `getMetadata()`, `getLinks()`, `getAudio()`, `getVideo()` があり、対応するフォーマットをリクエストした場合は追加のフィールドも含まれます。

<div id="simple-example">
  ### シンプルな例
</div>

```java
Document doc = client.scrape(
    "https://docs.firecrawl.dev",
    ScrapeOptions.builder().formats(List.of("markdown")).build()
);
```

<div id="complex-example">
  ### 複雑な例
</div>

```java
import com.firecrawl.models.ScrapeOptions;
import com.firecrawl.models.JsonFormat;

List<Map<String, Object>> actions = List.of(
    Map.of("type", "click", "selector", "#accept"),
    Map.of("type", "wait", "milliseconds", 750),
    Map.of("type", "scrape")
);

LocationConfig location = LocationConfig.builder()
    .country("US")
    .languages(List.of("en-US"))
    .build();

ScrapeOptions options = ScrapeOptions.builder()
    .formats(List.of(
        "markdown",
        "links",
        JsonFormat.builder().prompt("Extract plan names and prices.").build(),
        Map.of("type", "screenshot", "fullPage", true, "quality", 80)
    ))
    .onlyMainContent(true)
    .waitFor(1000)
    .parsers(List.of(Map.of("type", "pdf", "maxPages", 5)))
    .actions(actions)
    .location(location)
    .removeBase64Images(true)
    .blockAds(true)
    .proxy("auto")
    .maxAge(86400000L)
    .storeInCache(true)
    .build();

Document doc = client.scrape("https://example.com/pricing", options);
```

<div id="parameters">
  ### パラメータ
</div>

* `url`
  * Type: String
  * 使用する状況: 特定のページをスクレイピングしたい場合。

* `options.formats`
  * Type: フォーマット文字列またはフォーマットマップのリスト
  * 使用する状況: 複数の出力フォーマットが必要な場合。
  * 確認済みのフォーマット文字列:
    * `"markdown"`: markdownコンテンツ
    * `"html"`: 整形済みHTML
    * `"rawHtml"`: 生のHTML
    * `"links"`: ページリンク
    * `"images"`: 画像URL
    * `"screenshot"`: スクリーンショット出力
    * `"summary"`: 要約出力
    * `"changeTracking"`: changeTracking出力
    * `"json"`: JSON抽出
    * `"attributes"`: 属性抽出
    * `"branding"`: ブランドプロファイル出力
    * `"audio"`: 音声抽出
    * `"video"`: 動画抽出
  * フォーマットオブジェクトのフィールド:
    * `type`: 上記のフォーマット文字列のいずれか
    * `prompt`, `schema`: `type: "json"` のJSON抽出オプション
    * `modes`, `schema`, `prompt`, `tag`: `type: "changeTracking"` のchangeTrackingオプション
    * `fullPage`, `quality`, `viewport`: `type: "screenshot"` のスクリーンショットオプション
    * `selectors`: `type: "attributes"` の `{selector, attribute}` の配列

* `options.headers`
  * Type: `Map<String, String>`
  * 使用する状況: カスタムリクエストヘッダーが必要な場合。

* `options.includeTags`
  * Type: `List<String>`
  * 使用する状況: 特定のHTMLタグだけを含めたい場合。

* `options.excludeTags`
  * Type: `List<String>`
  * 使用する状況: 特定のHTMLタグを除外したい場合。

* `options.onlyMainContent`
  * Type: Boolean
  * 使用する状況: ナビゲーション、フッター、その他の定型コンテンツを取り除きたい場合。

* `options.timeout`
  * Type: Integer
  * 使用する状況: ミリ秒単位のタイムアウトが必要な場合。

* `options.waitFor`
  * Type: Integer
  * 使用する状況: ページのレンダリング完了を待つ必要がある場合 (ミリ秒) 。

* `options.mobile`
  * Type: Boolean
  * 使用する状況: モバイルビューポートを使用したい場合。

* `options.parsers`
  * Type: `List<Object>`
  * 使用する状況: ファイル解析の制御が必要な場合。
  * 確認済みの値:
    * `"pdf"`
    * `{type: "pdf", maxPages: number}`

* `options.actions`
  * Type: `List<Map<String, Object>>`
  * 使用する状況: 軽量なスクレイピング前アクションが必要な場合。
  * 確認済みのアクションタイプ:
    * `wait`: `milliseconds` または `selector` が必須
    * `screenshot`: `fullPage`, `quality`, `viewport` は任意
    * `click`: `selector` が必須
    * `write`: `text` が必須 (先にクリックして入力欄へフォーカスしてください)
    * `press`: `key` が必須
    * `scroll`: `direction` (`up` または `down`) が必須、`selector` は任意
    * `scrape`: 追加フィールドなし
    * `executeJavascript`: `script` が必須
    * `pdf`: `format` (A0, A1, A2, A3, A4, A5, A6, Letter, Legal, Tabloid, Ledger) 、`landscape`、`scale` は任意

* `options.location`
  * Type: `LocationConfig`
  * 使用する状況: 地理情報または言語を考慮したスクレイピングが必要な場合。

* `options.skipTlsVerification`
  * Type: Boolean
  * 使用する状況: TLS検証をスキップする必要がある場合。

* `options.removeBase64Images`
  * Type: Boolean
  * 使用する状況: markdown出力からbase64画像を除外したい場合。

* `options.blockAds`
  * Type: Boolean
  * 使用する状況: 広告やCookieポップアップをブロックしたい場合。

* `options.proxy`
  * Type: String
  * 使用する状況: プロキシの制御が必要な場合。
  * 確認済みの値: `"basic"`, `"stealth"`, `"enhanced"`, `"auto"`、またはカスタムプロキシURL文字列

* `options.maxAge`
  * Type: Long
  * 使用する状況: 指定した最大経過時間以内のキャッシュ済みデータを使いたい場合 (ミリ秒) 。

* `options.storeInCache`
  * Type: Boolean
  * 使用する状況: Firecrawl に結果をキャッシュさせたい場合。

* `options.integration`
  * Type: String
  * 使用する状況: APIがリクエストでインテグレーション識別子を必要とする場合。

<div id="interact">
  ## Interact
</div>

### 利用する理由

ページで、スクレイピング開始後にブラウザアクションやコードの実行が必要な場合は、Interact を使用します。

<div id="preferred-sdk-methods">
  ### 推奨SDKメソッド
</div>

* `client.interact(jobId, code)` — 既定の言語 `node` と、APIの既定の実行タイムアウトを使用します
* `client.interact(jobId, code, language, timeout)` — `timeout` は秒単位 (1～300) です。null を指定すると省略され、APIの既定値 (30秒) が使用されます
* `client.interact(jobId, code, language, timeout, origin)` — 任意の `origin` 文字列は、null でない場合にのみ送信されます (リクエスト元の識別)

<div id="simple-example">
  ### シンプルな例
</div>

```java
import com.firecrawl.models.BrowserExecuteResponse;

BrowserExecuteResponse result = client.interact(
    "<scrapeJobId>",
    "console.log(await page.title());",
    "node",
    60
);
```

<div id="complex-example">
  ### 複雑な例
</div>

```java
BrowserExecuteResponse result = client.interact(
    "<scrapeJobId>",
    "// ここでPlaywrightのページメソッドを使用してください",
    "node",
    120
);
```

<div id="stop-interactive-browser">
  ### 対話型ブラウザを停止
</div>

完了後、スクレイピングに紐づくブラウザセッションを終了します。

**推奨SDKメソッド:** `client.stopInteractiveBrowser(jobId)`

```java
import com.firecrawl.models.BrowserDeleteResponse;

BrowserDeleteResponse stopped = client.stopInteractiveBrowser("<scrapeJobId>");
```

<div id="interact-response-browserexecuteresponse">
  ### Interact レスポンス (`BrowserExecuteResponse`)
</div>

* `isSuccess()`: boolean
* `getStdout()`, `getStderr()`, `getResult()`, `getError()`: String (null の場合があります) 
* `getExitCode()`: Integer (null の場合があります) 
* `getKilled()`: Boolean (null の場合があります) — タイムアウトにより実行が停止された場合は true

<div id="stop-response-browserdeleteresponse">
  ### 停止時のレスポンス (`BrowserDeleteResponse`)
</div>

* `isSuccess()`: boolean
* `getSessionDurationMs()`: Long (null の場合があります) 
* `getCreditsBilled()`: Integer (null の場合があります) 
* `getError()`: String (null の場合があります)

<div id="parameters">
  ### パラメータ
</div>

* `jobId`
  * Type: String
  * Use when: スクレイピングジョブ ID がある場合。

* `code`
  * Type: String
  * Use when: ブラウザセッション内でコードを実行したい場合。

* `language`
  * Type: String
  * Use when: 特定のランタイムが必要な場合。
  * Confirmed values: `"python"`, `"node"`, `"bash"`
  * Notes: null の場合、SDK ではデフォルトで `"node"` が使用されます。

* `timeout`
  * Type: Integer
  * Use when: 秒単位の実行タイムアウト (1～300) が必要な場合。null の場合はこのフィールドは省略され、API のデフォルト値が使用されます。

* `origin`
  * Type: String
  * Use when: リクエストに任意の origin ラベルが必要な場合。インテグレーション識別子で必要な場合を除き、省略することを推奨します。

<div id="notes">
  ## 注記
</div>

* 非推奨のエイリアス: `scrapeExecute` → `interact`、`deleteScrapeBrowser` → `stopInteractiveBrowser` (および対応する `*Async` ヘルパー) 。
* Java SDK ではコードベースの操作のみをサポートしており、`interact` に `prompt` パラメータはありません (ほかの一部の言語 SDK とは異なります) 。
