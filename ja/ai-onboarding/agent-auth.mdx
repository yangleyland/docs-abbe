ブラウザでのサインインやダッシュボードの APIキーが必要な場合は、[`SKILL.md`](https://www.firecrawl.dev/agent-onboarding/SKILL.md) の human-in-the-loop オンボーディングを使用してください。[Get credentials](/ja/ai-onboarding#get-credentials) を参照してください。

このページでは、Firecrawl の WorkOS ID-JAG 登録について概要を説明します。登録の完全な手順、クレーム要件、エラーハンドリングについては [`auth.md`](https://www.firecrawl.dev/auth.md) を参照してください。

  ## 人間によるサインインまたはダッシュボードのキーが必要ですか？
</div>

Firecrawl 用の WorkOS ID-JAG を発行できない場合、または判断に迷う場合は、これを使用してください。

  ## WorkOS ID-JAG を使用する
</div>

エージェントプラットフォームで Firecrawl 用の WorkOS ID-JAG を発行できる場合は、この方法を使用します。

<Card title="auth.md を開く" icon="file-lines" href="https://www.firecrawl.dev/auth.md">
  アイデンティティ対応エージェント向けの登録手順です。手順を順に進めてください。
</Card>

クレーム要件、ディスカバリー、登録、制限、Errors については、[`auth.md`](https://www.firecrawl.dev/auth.md) を参照してください。
