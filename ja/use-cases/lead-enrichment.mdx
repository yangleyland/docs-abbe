ビジネスディレクトリをスクレイピングしてリードを見つけ、最新の企業データでCRMを強化し、アカウント調査を自動化できます。抽出はFirecrawlが処理するため、商談の成約に集中できます。

<div id="start-with-a-template">
  ## テンプレートから始める
</div>

<Card title="Fire Enrich" icon="github" href="https://github.com/firecrawl/fire-enrich">
  ウェブサイトからのAIによるリードエンリッチメントとデータ抽出
</Card>

<div id="how-it-works">
  ## 仕組み
</div>

1. **Firecrawl にデータソースを指定します。** ディレクトリの URL、企業サイト、またはドメインのリストを API に渡します。
2. **構造化データを抽出します。** Firecrawl は各ページをスクレイピングし、会社名、連絡先情報、チームメンバー、最新ニュースなどの項目を返します。
3. **データを CRM に送信します。** API または Zapier 連携を使って、エンリッチされたレコードを Salesforce、HubSpot、そのほかのツールに送信します。

<div id="why-sales-teams-choose-firecrawl">
  ## セールスチームが Firecrawl を選ぶ理由
</div>

<div id="extract-leads-from-any-directory">
  ### あらゆるディレクトリからリードを抽出
</div>

Firecrawlをビジネスディレクトリ、業界団体、またはカンファレンスの参加者リストに指定すると、企業の詳細情報や連絡先情報を含む構造化レコードが返され、そのままCRMにインポートできます。

<div id="keep-crm-data-fresh-automatically">
  ### CRMデータを自動で最新の状態に保つ
</div>

Firecrawlは静的なデータベースではなく、企業サイトから最新の情報を直接取得します。チームは常に最新の企業ニュース、人事異動、成長シグナルを把握できます。

<div id="lead-sources">
  ## リードの取得元
</div>

すでに手元にある情報に応じて、2種類の取得元からリードを取得できます。

<div id="business-directories">
  ### ビジネスディレクトリ
</div>

市場や業界内の企業一覧が必要な場合は、ディレクトリのURLを使用します。

* 業界別ディレクトリ
* 商工会議所の名簿
* 業界団体の会員名簿
* カンファレンス参加者名簿

<div id="company-websites">
  ### 企業サイト
</div>

対象アカウントがすでに分かっていて、アプローチに向けてより深い文脈が必要な場合は、企業サイトの URL を使用します。

* 会社概要ページやチーム紹介
* プレスリリースやニュース
* 成長シグナルを示す採用情報
* 導入事例

<div id="faqs">
  ## よくある質問
</div>

<AccordionGroup>
  <Accordion title="Firecrawl はどのようにリードエンリッチメントのプロセスを強化しますか？">
    Firecrawl は、見込み企業のウェブサイトから企業情報、連絡先、製品情報、最新ニュースを自動で抽出します。これにより、CRM が正確かつ最新の情報で充実し、より効果的なセールスアウトリーチが可能になります。
  </Accordion>

  <Accordion title="Firecrawl はウェブサイトから連絡先情報を見つけられますか？">
    はい。Firecrawl は、企業サイト、チームページ、問い合わせページから、メールや電話番号など公開されている連絡先情報を抽出します。
  </Accordion>

  <Accordion title="Firecrawl のリードデータはどの程度正確ですか？">
    Firecrawl は稼働中のウェブサイトから直接データを抽出するため、常に最新の情報を取得できます。これは、すぐに古くなる静的データベースよりも正確です。
  </Accordion>

  <Accordion title="自分の CRM と Firecrawl を連携できますか？">
    はい。API または Zapier 連携を使って、Salesforce、HubSpot、Pipedrive などの CRM のリードを自動でエンリッチできます。手作業のリサーチなしでリードデータを最新に保てます。
  </Accordion>

  <Accordion title="Firecrawl はアカウントベースドマーケティングにどのように役立ちますか？">
    ターゲットアカウントのウェブサイトから、詳細な企業情報、最新の更新、トリガーイベントを抽出します。これらのインサイトにより、アウトリーチのパーソナライズや、最適なエンゲージメントのタイミングの見極めが可能になります。
  </Accordion>
</AccordionGroup>
