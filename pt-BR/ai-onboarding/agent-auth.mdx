  ## O que acontece a seguir
</div>

Siga as etapas 1–4 em [`auth.md`](https://www.firecrawl.dev/auth.md) para entender o fluxo de registro, registrar-se em `POST /agent/auth` e usar a chave de API retornada com a API do Firecrawl.
