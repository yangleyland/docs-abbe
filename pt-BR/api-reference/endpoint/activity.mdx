title: 'Atividade'

Lista a atividade recente da sua API nas últimas 24 horas. Use isto para descobrir IDs de jobs e, em seguida, recuperar os resultados com o endpoint GET correspondente.

| Endpoint       | Endpoint de recuperação     |
| -------------- | --------------------------- |
| `scrape`       | `GET /v2/scrape/{id}`       |
| `crawl`        | `GET /v2/crawl/{id}`        |
| `batch_scrape` | `GET /v2/batch/scrape/{id}` |
| `agent`        | `GET /v2/extract/{id}`      |
