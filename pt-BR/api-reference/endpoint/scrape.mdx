<div id="interactions">
  ## Interações
</div>

Para interações no navegador (clicar, digitar, navegar, extrair conteúdo dinâmico), use o [endpoint Interact](/pt-BR/features/interact). Primeiro, faça o scraping da página e, em seguida, chame `POST /v2/scrape/{scrapeId}/interact` com um prompt em linguagem natural ou código Playwright para executar ações na página.

Consulte a [documentação do Interact](/pt-BR/features/interact) para ver todos os detalhes e exemplos.

Opcionalmente, você também pode usar o parâmetro `actions`, embora ele não seja recomendado para interações complexas.
