Adicione recursos de scraping e busca na web ao Claude.ai com o Firecrawl MCP usando conectores personalizados.

<Note>
  Procurando a configuração do Claude Code? Consulte o [guia do Claude Code](/pt-BR/quickstarts/claude-code).
</Note>

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenha sua chave de API
</div>

Cadastre-se em [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys) e copie sua chave de API.

<div id="2-add-custom-connector">
  ### 2. Adicionar conector personalizado
</div>

Acesse [Configurações &gt; Conectores](https://claude.ai/settings/connectors) no Claude.ai e clique em **Adicionar conector personalizado**.

Preencha os dados do conector:

* **URL:** `https://mcp.firecrawl.dev/YOUR_API_KEY/v2/mcp`
* **ID do cliente OAuth:** deixe em branco
* **Segredo do cliente OAuth:** deixe em branco

Substitua `YOUR_API_KEY` na URL pela sua [Firecrawl chave de API](https://www.firecrawl.dev/app/api-keys). A chave de API fica incorporada diretamente na URL, portanto não é necessário preencher campos adicionais de autenticação.

Clique em **Adicionar** para salvar o conector.

<Tip>
  **Prefere não colocar sua chave de API na URL?** Use o endpoint sem chave `https://mcp.firecrawl.dev/v2/mcp` no lugar (ainda deixando os campos OAuth em branco). O Claude.ai abrirá uma janela do navegador para que você faça login no Firecrawl e autorize o conector. Veja [Connect MCP Clients with OAuth](/pt-BR/developer-guides/mcp-setup-guides/oauth) para detalhes.
</Tip>

<div id="3-enable-in-conversation">
  ### 3. Ative na conversa
</div>

Em qualquer conversa no Claude.ai, clique no botão **+** no canto inferior esquerdo, acesse **Connectors** e ative o conector Firecrawl.

<div id="quick-demo">
  ## Demonstração rápida
</div>

Com o conector do Firecrawl habilitado, experimente estes prompts:

**Faça uma busca na web:**

```
Search for the latest Next.js 15 features
```

**Faça scraping de uma página:**

```
Scrape firecrawl.dev and tell me what it does
```

**Consulte a documentação:**

```
Find and scrape the Stripe API docs for payment intents
```

Claude usará automaticamente as ferramentas de busca e scraping da Firecrawl para obter informações.
