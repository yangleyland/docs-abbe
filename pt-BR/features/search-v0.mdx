<div id="searching-the-web-and-scraping-the-results-with-firecrawl">
  ## Pesquisando na web e extraindo os resultados com o Firecrawl
</div>

O Firecrawl integra sua API de busca à sua infraestrutura robusta de scraping para oferecer, em um único endpoint, uma experiência contínua de busca e extração. Veja por quê:

1. **Consulta de busca unificada:**
   Os usuários enviam uma consulta de busca pelo endpoint de busca.

2. **Extração automatizada dos resultados:**
   O Firecrawl processa automaticamente os resultados da busca e utiliza seus recursos de scraping para extrair dados de cada página de resultado.

3. **Entrega dos dados:**
   Os dados extraídos de todas as páginas de resultado são compilados e entregues em markdown limpo — pronto para uso.

Essa integração permite realizar buscas na web com eficiência e obter dados abrangentes, já raspados de múltiplas fontes, com esforço mínimo.

Para mais detalhes, consulte a [documentação do endpoint /search](https://docs.firecrawl.dev/api-reference/endpoint/search).

<div id="search-endpoint">
  ### endpoint /search
</div>

Usado para pesquisar na web, obter os resultados mais relevantes, extrair cada página e retornar o markdown.

```bash
curl -X POST https://api.firecrawl.dev/v0/search \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "query": "firecrawl",
      "pageOptions": {
        "fetchPageContent": true // false para uma API de SERP mais rápida
      }
    }'
```

```json
{
  "success": true,
  "data": [
    {
      "url": "https://docs.firecrawl.dev",
      "markdown": "# Conteúdo em Markdown",
      "provider": "web-scraper",
      "metadata": {
        "title": "Firecrawl | Raspagem da web confiável para seus LLMs",
        "description": "IA para CX e Vendas"
        "language": null,
        "sourceURL": "https://docs.firecrawl.dev/"
      }
    }
  ]
}
```

<div id="with-python-sdk">
  ### Com o SDK em Python
</div>

<div id="installing-python-sdk">
  #### Instalando o SDK do Python
</div>

```bash
pip install firecrawl-py
```

#### Executar uma consulta

```python
from firecrawl import Firecrawl

app = Firecrawl(api_key="YOUR_API_KEY")

result = app.search(query="What is firecrawl?")
```

A resposta será semelhante à do comando curl acima.

<div id="search-a-query">
  #### Pesquisar uma consulta
</div>

```go
import (
  "fmt"
  "log"

  "github.com/mendableai/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
      log.Fatalf("Failed to initialize Firecrawl: %v", err)
  }

  query := "What is firecrawl?"
  searchResult, err := app.Search(query)
  if err != nil {
    log.Fatalf("Failed to search: %v", err)
  }
  fmt.Println(searchResult)
}
```

```toml
serde_json = "^1.0"

[build-dependencies]
tokio = { version = "1", features = ["full"] }
```

#### Fazer uma busca

```rust
async fn main() {
  let api_key = "YOUR_API_KEY";
  let api_url = "https://api.firecrawl.dev";
  let app = Firecrawl::new(api_key, api_url).expect("Failed to initialize Firecrawl");

  let query = "What is firecrawl?";
  let search_result = app.search(query).await;

  match search_result {
    Ok(data) => println!("Search Result: {}", data),
    Err(e) => eprintln!("Failed to search: {}", e),
  }
}
```
