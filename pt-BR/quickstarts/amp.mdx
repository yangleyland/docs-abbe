Adicione as ferramentas de busca, scraping, rastreamento e navegador do Firecrawl ao [Sourcegraph Amp](https://ampcode.com) via MCP.

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenha sua chave de API
</div>

Cadastre-se em [firecrawl.dev/app](https://www.firecrawl.dev/app) e copie sua chave de API.

<div id="2-add-firecrawl-to-amp">
  ### 2. Adicione o Firecrawl ao Amp
</div>

Abra as configurações do Amp e adicione um servidor MCP. O Amp aceita a configuração padrão do MCP:

```json
{
  "amp.mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Substitua `fc-YOUR-API-KEY` pela sua chave de API do Firecrawl.

<div id="3-reload-amp">
  ### 3. Recarregue o Amp
</div>

Recarregue a janela do Amp. As ferramentas do Firecrawl agora estão disponíveis para o agente.

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hospedada remotamente (sem precisar de Node.js)
</div>

```json
{
  "amp.mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Solução de problemas
</div>

* **O servidor não inicia** — consulte a exibição de logs do MCP do Amp para ver a saída de stderr.
* **`npx` ausente** — instale o Node.js 18+ ou use a URL hospedada remotamente acima.
