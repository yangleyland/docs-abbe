Adicione as ferramentas de busca, scraping, rastreamento e navegador do Firecrawl ao [Google Antigravity](https://antigravity.google/) via MCP.

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenha sua chave de API
</div>

Cadastre-se em [firecrawl.dev/app](https://www.firecrawl.dev/app) e copie sua chave de API.

<div id="2-add-firecrawl-to-antigravity">
  ### 2. Adicione o Firecrawl ao Antigravity
</div>

Abra as configurações do Antigravity (`Cmd/Ctrl + ,`), procure por **MCP Servers** e adicione um novo servidor com:

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Substitua `fc-YOUR-API-KEY` pela sua chave de API do Firecrawl.

<div id="3-reload-antigravity">
  ### 3. Recarregue o Antigravity
</div>

Recarregue a janela (`Cmd/Ctrl + Shift + P` → `Reload Window`). O agente agora tem acesso às ferramentas web do Firecrawl.

<div id="quick-demo">
  ## Demonstração rápida
</div>

Em um chat com um agente do Antigravity:

```
Search the web for "Vercel AI SDK v5 release notes" and summarize.
```

```
Scrape https://docs.firecrawl.dev/ai-onboarding and list every linked guide.
```

```
Crawl https://example.com and extract every page title.
```

O Antigravity encaminha automaticamente essas chamadas de ferramentas pelo MCP do Firecrawl.

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hospedada remotamente (sem precisar de Node.js)
</div>

```json
{
  "mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Solução de problemas
</div>

* **Servidor aparece como &quot;failed&quot;** — verifique o painel de saída do MCP para `stderr`. Na maioria dos casos, a falha ocorre por falta de uma chave de API ou porque o `npx` não está no `PATH`.
* **Ferramentas não invocadas** — mencione explicitamente o Firecrawl (por exemplo, &quot;Use Firecrawl para fazer scraping…&quot;) no primeiro prompt para que o agente escolha a ferramenta certa.
