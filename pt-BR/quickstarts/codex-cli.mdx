Adicione as ferramentas de busca, scraping, rastreamento e navegador do Firecrawl ao [OpenAI Codex CLI](https://github.com/openai/codex) via MCP.

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenha sua chave de API
</div>

Cadastre-se em [firecrawl.dev/app](https://www.firecrawl.dev/app) e copie sua chave de API.

<div id="2-add-firecrawl-to-codex">
  ### 2. Adicione o Firecrawl ao Codex
</div>

O Codex lê a configuração do servidor MCP em `~/.codex/config.toml`. Adicione o servidor Firecrawl:

```toml
[mcp_servers.firecrawl]
command = "npx"
args = ["-y", "firecrawl-mcp"]

[mcp_servers.firecrawl.env]
FIRECRAWL_API_KEY = "fc-YOUR-API-KEY"
```

Substitua `fc-YOUR-API-KEY` pela sua chave de API do Firecrawl.

<div id="3-start-codex">
  ### 3. Iniciar o Codex
</div>

```bash
codex
```

O Codex detecta as ferramentas do Firecrawl ao iniciar. Confirme se elas foram carregadas:

```bash
/mcp
```

Você deverá ver `firecrawl` listado com ferramentas como `firecrawl_search`, `firecrawl_scrape`, `firecrawl_crawl` e `firecrawl_extract`.

<div id="quick-demo">
  ## Demonstração rápida
</div>

Experimente estes prompts:

```
Search the web for the latest Next.js App Router release notes and summarize.
```

```
Scrape https://docs.firecrawl.dev and list the top-level sections.
```

```
Crawl https://example.com and save the markdown for every page under /blog.
```

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hospedada remotamente (sem necessidade de Node.js)
</div>

Se preferir não executar `npx` localmente:

```toml
[mcp_servers.firecrawl]
url = "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
```

<div id="troubleshooting">
  ## Solução de problemas
</div>

* **O Codex não reconhece as ferramentas** — execute `codex --version` para confirmar que você está usando uma versão com suporte a MCP e, em seguida, reinicie a CLI após editar `config.toml`.
* **`spawn npx ENOENT`** — instale o Node.js 18+ e verifique se `npx` está no seu `PATH`, ou use a URL hospedada remotamente acima.
* **401 / chave inválida** — gere uma nova chave de API em [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys).
