Adicione recursos de raspagem e pesquisa na web ao Cursor com o Firecrawl MCP.

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenha sua chave de API
</div>

Crie uma conta em [firecrawl.dev/app](https://firecrawl.dev/app) e copie sua chave de API.

<div id="2-add-to-cursor">
  ### 2. Adicionar ao Cursor
</div>

Abra as configurações (`Cmd+,`), pesquise por &quot;MCP&quot; e adicione:

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "sua_chave_de_api_aqui"
      }
    }
  }
}
```

Substitua `your_api_key_here` pela sua chave de API do Firecrawl.

<div id="3-restart-cursor">
  ### 3. Reinicie o Cursor
</div>

Pronto! Agora você pode pesquisar e raspar a web diretamente no Cursor.

<div id="quick-demo">
  ## Demo Rápido
</div>

Experimente isto no Cursor Chat (`Cmd+K`):

**Pesquisar:**

```
Buscar melhores práticas de TypeScript 2025
```

**Raspagem:**

```
Faça scraping do firecrawl.dev e me diga o que ele faz
```

**Obter documentação:**

```
Faça scraping da documentação de hooks do React e explique o useEffect
```

O Cursor usará automaticamente as ferramentas do Firecrawl.


<div id="windows-troubleshooting">
  ## Solução de problemas no Windows
</div>

Se você encontrar um erro `spawn npx ENOENT` ou &quot;No server info found&quot; no Windows, o Cursor não está encontrando `npx` no seu PATH. Tente uma destas soluções:

**Opção A: Use o caminho completo para `npx.cmd`**

Execute `where npx` no Prompt de Comando para obter o caminho completo e, em seguida, atualize sua configuração:

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "C:\\Program Files\\nodejs\\npx.cmd",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "your_api_key_here"
      }
    }
  }
}
```

Substitua o caminho do `command` pela saída de `where npx`.

**Opção B: Use a URL hospedada remotamente (sem Node.js)**

```json
{
  "mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/YOUR-API-KEY/v2/mcp"
    }
  }
}
```

Substitua `YOUR-API-KEY` pela sua chave de API do Firecrawl.
