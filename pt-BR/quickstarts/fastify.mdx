<div id="prerequisites">
  ## Pré-requisitos
</div>

* Node.js 18+
* Uma chave de API do Firecrawl — [obtenha uma grátis](https://www.firecrawl.dev/app/api-keys)

<div id="setup">
  ## Configuração
</div>

```bash
npm install fastify firecrawl
```

Adicione sua chave de API ao arquivo `.env`:

```bash
FIRECRAWL_API_KEY=fc-YOUR-API-KEY
```

<div id="search-the-web">
  ## Fazer uma busca na web
</div>

```javascript
import Fastify from "fastify";
import { Firecrawl } from "firecrawl";

const fastify = Fastify({ logger: true });
const firecrawl = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

fastify.post("/search", async (request) => {
  const { query } = request.body;
  return firecrawl.search(query, { limit: 5 });
});

fastify.listen({ port: 3000 });
```

<div id="scrape-a-page">
  ## Fazer scraping de uma página
</div>

```javascript
fastify.post("/scrape", async (request) => {
  const { url } = request.body;
  return firecrawl.scrape(url);
});
```

<div id="interact-with-a-page">
  ## Interagir com uma página
</div>

Use o interact para controlar uma sessão ativa do navegador — clique em botões, preencha formulários e extraia conteúdo dinâmico.

```javascript
fastify.post("/interact", async (request) => {
  const { url } = request.body;

  const result = await firecrawl.scrape(url, { formats: ['markdown'] });
  const scrapeId = result.metadata?.scrapeId;

  await firecrawl.interact(scrapeId, { prompt: 'Search for iPhone 16 Pro Max' });
  const response = await firecrawl.interact(scrapeId, { prompt: 'Click on the first result and tell me the price' });

  await firecrawl.stopInteraction(scrapeId);

  return { output: response.output };
});
```

<div id="as-a-fastify-plugin">
  ## Como plugin do Fastify
</div>

Encapsule o cliente em um plugin para reutilizá-lo em várias rotas:

```javascript
import fp from "fastify-plugin";
import { Firecrawl } from "firecrawl";

export default fp(async function firecrawlPlugin(fastify) {
  const client = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });
  fastify.decorate("firecrawl", client);
});
```

Registre o plugin e, em seguida, use `fastify.firecrawl` em qualquer rota:

```javascript
fastify.register(firecrawlPlugin);

fastify.post("/search", async function (request) {
  const { query } = request.body;
  return this.firecrawl.search(query, { limit: 5 });
});
```
