Adicione as ferramentas de busca, scraping, rastreamento e navegador do Firecrawl ao [Google Gemini CLI](https://github.com/google-gemini/gemini-cli) via MCP.

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenha sua chave de API
</div>

Cadastre-se em [firecrawl.dev/app](https://www.firecrawl.dev/app) e copie sua chave de API.

<div id="2-add-firecrawl-to-gemini-cli">
  ### 2. Adicione o Firecrawl ao Gemini CLI
</div>

O Gemini CLI lê a configuração do MCP em `~/.gemini/settings.json` (global) ou em `.gemini/settings.json` no seu projeto. Adicione:

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Substitua `fc-YOUR-API-KEY` pela sua chave de API do Firecrawl.

<div id="3-launch-gemini-cli">
  ### 3. Inicie o Gemini CLI
</div>

```bash
gemini
```

Verifique se o servidor está em execução:

```
/mcp list
```

Você verá `firecrawl` e suas ferramentas.

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hospedada remotamente (sem necessidade de Node.js)
</div>

```json
{
  "mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Solução de problemas
</div>

* **As ferramentas não aparecem** — reinicie o Gemini CLI após editar `settings.json`; os servidores MCP são carregados ao iniciar.
* **`spawn npx ENOENT`** — instale o Node.js 18+ ou use a URL hospedada remotamente.
* **Limite de taxa atingido** — atualize seu plano Firecrawl em [firecrawl.dev/pricing](https://www.firecrawl.dev/pricing).
