Adicione as ferramentas de busca, scraping, rastreamento e navegador do Firecrawl ao [OpenCode](https://opencode.ai) via MCP.

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenha sua chave de API
</div>

Cadastre-se em [firecrawl.dev/app](https://www.firecrawl.dev/app) e copie sua chave de API.

<div id="2-add-firecrawl-to-opencode">
  ### 2. Adicione o Firecrawl ao OpenCode
</div>

O OpenCode lê a configuração em `~/.config/opencode/config.json` (global) ou em `./opencode.json` no seu projeto. Adicione:

```json
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "firecrawl": {
      "type": "local",
      "command": ["npx", "-y", "firecrawl-mcp"],
      "environment": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Substitua `fc-YOUR-API-KEY` pela sua chave da API do Firecrawl.

<div id="3-start-opencode">
  ### 3. Inicie o OpenCode
</div>

```bash
opencode
```

O OpenCode carrega os servidores MCP ao iniciar. Confirme que o Firecrawl está vinculado:

```
/mcp
```

<div id="remote-hosted-url-no-nodejs-required">
  ## URL hospedada remotamente (sem necessidade de Node.js)
</div>

```json
{
  "mcp": {
    "firecrawl": {
      "type": "remote",
      "url": "https://mcp.firecrawl.dev/fc-YOUR-API-KEY/v2/mcp"
    }
  }
}
```

<div id="troubleshooting">
  ## Solução de problemas
</div>

* **Servidor não conectado** — execute `opencode doctor` para verificar erros de carregamento do MCP.
* **Permissão negada no `npx`** — instale o Node.js 18+ e verifique se seu shell reconhece a instalação (`which npx`).
