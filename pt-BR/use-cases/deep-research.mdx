<div id="how-it-works">
  ## Como funciona
</div>

Crie ferramentas de pesquisa poderosas que transformam dados dispersos da web em insights abrangentes. O padrão central é um loop de **busca → scraping → análise → repetição**: use a API de busca do Firecrawl para descobrir fontes relevantes, faça o scraping de cada fonte para obter o conteúdo completo e, em seguida, envie os resultados para um LLM sintetizar as descobertas e identificar consultas de acompanhamento.

<Steps>
  <Step title="Buscar fontes">
    Use o endpoint `/search` para encontrar páginas relevantes para seu tópico de pesquisa.

    <CodeGroup>
      ```python Python
      from firecrawl import Firecrawl

      firecrawl = Firecrawl(api_key="fc-YOUR-API-KEY")

      results = firecrawl.search(
          "recent advances in quantum computing",
          limit=5,
          scrape_options={"formats": ["markdown", "links"]}
      )
      ```

      ```js Node.js
      import { Firecrawl } from 'firecrawl';

      const firecrawl = new Firecrawl({ apiKey: "fc-YOUR-API-KEY" });

      const results = await firecrawl.search(
        'recent advances in quantum computing',
        { limit: 5, scrapeOptions: { formats: ['markdown', 'links'] } }
      );
      ```
    </CodeGroup>
  </Step>

  <Step title="Fazer scraping das páginas encontradas">
    Extraia o conteúdo completo de cada resultado para obter informações detalhadas com citações.

    <CodeGroup>
      ```python Python
      for result in results:
          doc = firecrawl.scrape(result["url"], formats=["markdown"])
          # Envie o conteúdo do doc para seu LLM para análise
      ```

      ```js Node.js
      for (const result of results) {
        const doc = await firecrawl.scrape(result.url, { formats: ['markdown'] });
        // Envie o conteúdo do doc para seu LLM para análise
      }
      ```
    </CodeGroup>
  </Step>

  <Step title="Analisar e iterar">
    Use um LLM para sintetizar descobertas, identificar lacunas e gerar consultas de acompanhamento. Repita o loop até que sua pergunta de pesquisa seja respondida por completo.
  </Step>
</Steps>

  ## Por que pesquisadores escolhem o Firecrawl
</div>

<div id="accelerate-research-from-weeks-to-hours">
  ### Acelere a pesquisa: de semanas para horas
</div>

Crie sistemas de pesquisa automatizados que descubram, leiam e sintetizem informações de toda a web. Desenvolva ferramentas que entreguem relatórios abrangentes com citações completas, eliminando a necessidade de buscar manualmente entre centenas de fontes.

<div id="ensure-research-completeness">
  ### Garanta a Completude da Pesquisa
</div>

Reduza o risco de deixar passar informações críticas. Crie sistemas que sigam cadeias de citações, descubram fontes relacionadas e revelem insights que os métodos tradicionais de busca não capturam.

<div id="faqs">
  ## FAQs
</div>

<AccordionGroup>
  <Accordion title="Como posso criar ferramentas de pesquisa com o Firecrawl?">
    Use as APIs de crawl e search do Firecrawl para construir sistemas de pesquisa iterativos. Comece com resultados de busca, extraia conteúdo de páginas relevantes, siga links de citação e agregue as descobertas. Combine com LLMs para sintetizar relatórios de pesquisa abrangentes.
  </Accordion>

  <Accordion title="O Firecrawl consegue lidar com sites acadêmicos e científicos?">
    Sim. O Firecrawl pode extrair dados de artigos de acesso aberto, sites acadêmicos e publicações científicas disponíveis ao público. Ele preserva a formatação, as citações e o conteúdo técnico essenciais para trabalhos de pesquisa.
  </Accordion>

  <Accordion title="Como garanto a precisão dos dados de pesquisa?">
    O Firecrawl mantém a atribuição de fonte e extrai o conteúdo exatamente como apresentado nos sites. Todos os dados incluem URLs de origem e carimbos de data e hora, garantindo total rastreabilidade para fins de pesquisa.
  </Accordion>

  <Accordion title="Posso usar o Firecrawl para estudos longitudinais?">
    Sim. Configure crawls agendados para acompanhar como as informações mudam ao longo do tempo. Isso é ideal para monitorar tendências, mudanças em políticas públicas ou qualquer pesquisa que exija análise temporal de dados.
  </Accordion>

  <Accordion title="Como o Firecrawl lida com projetos de pesquisa em grande escala?">
    Nossa infraestrutura de crawling escala para lidar com milhares de fontes simultaneamente. Seja analisando setores inteiros ou acompanhando tendências globais, o Firecrawl fornece o pipeline de dados de que você precisa.
  </Accordion>
</AccordionGroup>
