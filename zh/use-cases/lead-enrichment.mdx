<div id="how-it-works">
  ## 工作方式
</div>

1. **将数据源提供给 Firecrawl。** 向 API 传入目录 URL、公司网站或域名列表。
2. **提取结构化数据。** Firecrawl 会抓取每个页面，并返回公司名称、联系方式、团队成员和近期新闻等字段。
3. **将数据发送到你的 CRM。** 通过 API 或 Zapier 集成，将补全后的记录推送到 Salesforce、HubSpot 或任何其他工具。

  ## 为什么销售团队选择 Firecrawl
</div>

<div id="extract-leads-from-any-directory">
  ### 从任何名录中提取 线索
</div>

将 Firecrawl 用于企业名录、行业协会或会议参会者名单，即可返回包含公司详情和联系信息的结构化记录，可直接导入 CRM。

<div id="keep-crm-data-fresh-automatically">
  ### 自动保持 CRM 数据最新
</div>

Firecrawl 直接从公司官网获取实时信息，而非依赖静态数据库。你的团队始终掌握最新的公司动态、人员变动和增长信号。

<div id="lead-sources">
  ## 线索来源
</div>

根据你手头已有的信息，从两类来源中获取线索。

<div id="business-directories">
  ### 商业名录
</div>

当你需要获取某个市场或行业的公司名单时，请使用名录 URL。

* 行业专项名录
* 商会名录
* 行业协会成员名单
* 会议参会者名单

<div id="company-websites">
  ### 公司网站
</div>

当你已经了解该账户，并希望为外联获取更深入的背景信息时，请使用公司 URL。

* 关于页面与团队栏目
* 新闻稿与新闻动态
* 招聘信息 (增长信号)
* 客户案例

<div id="faqs">
  ## 常见问题
</div>

<AccordionGroup>
  <Accordion title="Firecrawl 如何提升线索富集流程？">
    Firecrawl 会自动从潜在客户网站提取公司信息、联系方式、产品与服务以及最新动态。它可将准确、最新的信息写入你的 CRM，提升销售外联效果。
  </Accordion>

  <Accordion title="Firecrawl 能从网站找到联系信息吗？">
    可以！Firecrawl 会从公司官网、团队页面和联系页面提取公开的联系信息，包括邮箱和电话号码。
  </Accordion>

  <Accordion title="Firecrawl 提供的线索数据有多准确？">
    由于 Firecrawl 直接从在线网站提取数据，你获得的是当前最新的信息。这比容易过时的静态数据库更为准确。
  </Accordion>

  <Accordion title="我可以将 Firecrawl 集成到我的 CRM 吗？">
    可以。使用我们的 API 或 Zapier 集成，可在 Salesforce、HubSpot、Pipedrive 及其他 CRM 中自动富集线索数据。无需手动调研即可保持线索数据新鲜。
  </Accordion>

  <Accordion title="Firecrawl 如何助力基于账户的营销（ABM）？">
    从目标账户网站提取详细的公司信息、最新动态和触发事件。这些情报有助于个性化外联，并识别最佳触达时机。
  </Accordion>
</AccordionGroup>
